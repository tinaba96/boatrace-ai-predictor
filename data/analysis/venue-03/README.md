## 重要な発見

1. **信頼度70-79%が80%以上より高回収率**
   - 高信頼度レースは配当が低い
   - 中程度の信頼度で配当妙味を狙う

2. **2号艇は前半レース(1-4R)で狙う**
   - 前半: 147.7%、後半: 27.5%

3. **3号艇は中盤・後半レースで狙う**
   - 中盤: 122.5%、後半: 128.9%

4. **1号艇がB級の時は2号艇予測が有効**
   - インが弱い時の外枠狙い

5. **3連複は極めて高回収率**
   - 1-2-4組み合わせ: 610.7%（73R）
   - 1-2-4×後半レース: 1107.6%（21R）
   - 1号艇含む予測全体: 383.5%（409R）

6. **3連複は後半レース（9-12R）が特に有効**
   - 後半レースで回収率が1.8倍に上昇

7. **3連単は非推奨（深掘り分析で確認）**
   - 的中率: 5.8%、回収率: 19.8%
   - 3連複（397.5%）と比較して大幅に劣る
   - **3連複的中時の3連単的中率: 32.9%**（組み合わせが当たっても順番は1/3）
   - **1着的中時の2-3着順番的中率: 12.7%**
   - **高配当(3000円↑)的中: 0件**
   - 唯一の有望条件: 1-3-2×信頼度80↑×後半(9-12R) = 105.4%(13R)
