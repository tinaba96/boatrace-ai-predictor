## 重要な発見

1. **3号艇・5号艇の単勝が高回収率**
   - 3号艇1着+4号艇含む: 199%（29R）
   - 5号艇1着+1号艇含む: 190%（25R）

2. **複勝は4号艇・5号艇1着が有効**
   - 4号艇1着×conf75+: 135%（41R）
   - 5号艇1着+1号艇含む: 134%（25R）

3. **1号艇勝率は高いが回収率は低い**
   - 1号艇勝率57.4%だが、単勝回収率は100%未満
   - 穴狙い（3〜5号艇）が有効
