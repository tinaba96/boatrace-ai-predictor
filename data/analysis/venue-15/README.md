## 重要な発見

1. **5号艇1着が非常に有効**
   - 5号艇1着+2号艇含む: 222%（25R）複勝
   - 5号艇1着+1号艇含む: 205%（30R）単勝
   - 5号艇1着: 147%（56R）複勝

2. **6号艇1着の単勝も高回収率**
   - 6号艇1着+3号艇含む: 236%（22R）
   - 6号艇1着+1号艇含む: 153%（35R）

3. **3連複は狙わない**
   - 有効なルールが見つからず
