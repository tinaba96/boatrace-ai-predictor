分析期間: 過去180日（2025-11-01〜2026-04-30）
対象レース: 20,154件

分析スクリプト:
- `scripts/analysis/redesign-volatility-threshold.js` — 全体閾値分析（90日）
- `scripts/analysis/redesign-volatility-by-venue.js` — 会場別分析（180日）

## 問題の整理

### 問題1: 共通閾値55が不適切

会場ごとに1コース飛び率（ベースライン）が大きく異なる。

| グループ | 会場 | ベース飛び率 |
|---------|------|------------|
| 高飛び（≥55%） | 戸田、平和島 | 55〜59% |
| 標準（42〜55%） | 江戸川、鳴門、桐生 等16会場 | 43〜55% |
| 堅い（<42%） | 徳山、住之江、芦屋、若松、尼崎、大村 | 39〜41% |

同一閾値55では、元々飛びやすい会場で「high」が乱発され、
堅い会場では真のイン崩れを取りこぼす。

### 問題2: 55-59点帯が無意味

現行highの下限付近（55-59点、636件）の飛び率は **48.9%（+2.2pt）**。
ベースライン46.7%と実質変わらない。

| スコア帯 | 飛び率 | ベース差 |
|---------|--------|---------|
| 55-59点 | 48.9% | +2.2pt（無意味） |
| 60-64点 | 53.9% | +7.1pt |
| 65-69点 | 54.9% | +8.1pt |
| 70-74点 | 58.5% | +11.7pt |
| 75-79点 | 58.5% | +11.8pt |
| 80-84点 | 59.3% | +12.6pt |

### 問題3: 無効特徴量の混在

| 特徴量 | スピアマン相関（全体） | 判定 |
|--------|-----------------|------|
| 1号艇全国勝率 | ρ=0.235 | **残す（主軸）** |
| 1号艇avgST | ρ=0.239 | **残す（主軸）** |
| 選手間σ | ρ=0.091 | 重みを大幅縮小 |
| モーター2連率 | ρ=0.093 | **除外** |

モーター2連率は全会場でρ<0.20（ほぼ全会場で<0.10）。
スコアを希釈するだけなので除外が妥当。

## 注意事項

- 戸田(02)はどの閾値でも+10pt達成困難。高飛び会場のためhighラベルの価値が薄い。score≥80を閾値とするが、ユーザーには「元々飛びやすい会場」の注釈が必要
- avgSTはデータ欠損が多い会場あり（スクレイピング取得率の影響）。欠損時は全国勝率に寄せる処理が必要
