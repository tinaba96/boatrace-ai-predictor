# 新概念データ・超展開データの詳細分析

## 概要

ボートレース日和（kyoteibiyori.com）が提供するオリジナルデータ群。公式サイト（boatrace.jp）には存在しない独自分析データで、1マークの展開予想を精度高く行うために設計されている。

大きく分けて以下の3系統がある。

| データ系統 | 初出時期 | 概要 |
|-----------|---------|------|
| 決まり手データ（新概念データ） | 2020年8月 | 選手の枠別決まり手率を数値化。「勝ち方」と「負け方」の両面を分析 |
| 超展開データ | 2025年9月 | 1マークの攻め方・攻められ方を可視化。抵抗・絞りの回数も計測 |
| オリジナル展示タイム | 2024年9月 | 周回タイム・展示タイムの独自分析。展示ST vs 本番STのズレ評価 |

これらはすべて出走表の「枠別情報」タブおよびMyDataから確認できる。

---

## 決まり手データの詳細

### 定義

決まり手データとは、**選手が該当コースでどのようにして勝ったかを数値化したもの**。ボートレースの決まり手は全6種類あり、それぞれの発生率を選手×コース別に集計している。

### 決まり手の種類

| 決まり手 | 説明 | 主に該当するコース |
|---------|------|------------------|
| 逃げ | 1マークを先頭で回り、そのまま1着 | 1コース |
| 差し | 内側の艇のターンの内側を突いて1着 | 2・3・4コース |
| まくり（捲り） | 外側から全速ターンで内側艇を一気に抜く | 2・3・4・5・6コース |
| まくり差し（捲り差し） | まくった後、内側に差し込む複合技 | 3・4・5・6コース |
| 抜き | 2周目以降に追い抜いて1着 | 全コース |
| 恵まれ | 先行艇の転覆・失格等により繰り上がり | 全コース |

### 数値化の方法

各選手について、コース別に「そのコースで1着を取った際の決まり手の割合（%）」を算出する。

**表示例（1号艇・1コース）:**

| 項目 | 数値例 |
|------|--------|
| 逃げ率 | 48% |
| 差し率 | 0% |
| まくり率 | 0% |

**表示例（4号艇・4コース）:**

| 項目 | 数値例 |
|------|--------|
| 差し率 | 35% |
| まくり率 | 25% |
| まくり差し率 | 20% |
| 抜き率 | 15% |
| 恵まれ率 | 5% |

### グレード別切り替え

4種類のグレードカテゴリでデータを切り替えて閲覧可能。

| カテゴリ | 対象レース |
|---------|-----------|
| 総合 | すべてのレースの集計 |
| 一般戦 | 一般戦、G2、G3レースの集計 |
| SG・G1 | SG、G1レースの集計 |
| 女子戦 | 女子専用レース（レディースチャンピオン等）の集計 |

**切り替えの意義**: 同じ選手でもグレードによって走りが変わる。SG・G1ではA1選手ばかりなので、一般戦とは展開が大きく異なる。女子戦は体重差が少なく独特の傾向がある。

### 集計期間

- **直近6ヶ月**: 最近の調子・傾向を把握
- **直近1年**: より安定した統計データ

両方を確認し、傾向が一致しているか、最近変化しているかを判断する。

### MyDataへの登録

以下の項目をMyDataに登録してカスタマイズ表示が可能。

| MyData項目名 | 内容 |
|-------------|------|
| 枠別｜決まり手（総合）｜上：直近1年｜下：直近6ヶ月 | 決まり手率の時系列比較 |
| 枠別｜決まり手数（総合）｜直近1年 | 決まり手の絶対数（サンプル数確認用） |

## 新概念データの詳細

### 定義

新概念データとは、**各選手の枠別の決まり手率を元に「走りのクセ」を数値化したデータ**。従来の「勝ち方」だけでなく「負け方」にも着目した点が革新的。BOATBoy（月刊誌）が考案した概念をボートレース日和がデータ化した。

### 従来の決まり手データとの違い

| 観点 | 従来の決まり手データ | 新概念データ |
|------|-------------------|------------|
| 着目点 | 勝った時の決まり手 | 勝ちも負けも含めた走りのクセ |
| 1コースのデータ | 逃げ率のみ | 逃げ率 + 差され率 + まくられ率 + まくり差され率 |
| 2コースのデータ | 差し率・まくり率 | 差し率・まくり率 + 逃し率 |
| 3〜6コースのデータ | 各決まり手率 | 各決まり手率 + 逃し率・差され率・まくられ率・まくり差され率 |

### 具体的な項目

#### 1コース（インコース）の新概念データ

| 項目 | 定義 | 活用方法 |
|------|------|---------|
| 逃げ率（1着率） | 1コースから逃げて1着を取った割合 | インの信頼度の基本指標 |
| 差され率 | 1コースで負けた際に差しで負けた割合 | 2・3コースからの差しに弱いかの判断 |
| まくられ率 | 1コースで負けた際にまくりで負けた割合 | スタートの遅さ・外からの攻めへの耐性 |
| まくり差され率（捲れ差率） | 1コースで負けた際にまくり差しで負けた割合 | 外枠からの複合攻撃への耐性 |

**活用例**: まくられ率がゼロの選手はスタートが上手く、インコースのアドバンテージを最大限に活用できる選手。

#### 2コース（差しの定位置）の新概念データ

| 項目 | 定義 | 活用方法 |
|------|------|---------|
| 差し率 | 2コースから差して1着を取った割合 | 2コースの攻撃力 |
| まくり率 | 2コースからまくって1着を取った割合 | 2コースの攻撃スタイル |
| 逃し率（逃がし率） | 2コースが1号艇のイン逃げを許した割合 | イン逃げの信頼度判断に重要 |

**活用例**: 1号艇の逃げ率 + 2号艇の逃がし率が100%を超えていて、かつ外枠の決まり手率が10%を超えない場合、イン逃げが信頼できる。

#### 3〜6コースの新概念データ

2024年5月のバージョンアップで、2〜6コースすべてに以下の「負け方データ」が追加された。

| 項目 | 定義 |
|------|------|
| 逃し率 | そのコースの選手が1号艇のイン逃げを許した割合 |
| 差され率 | そのコースの選手が差しで負けた割合 |
| まくられ率 | そのコースの選手がまくりで負けた割合 |
| 捲れ差率（まくり差され率） | そのコースの選手がまくり差しで負けた割合 |

### 1コースが負けた時の決まり手一覧

2024年5月に追加された機能。1コースの選手が1着を取れなかった場合、**どのコースの選手にどの決まり手で負けたか**を一覧表示する。

**表示形式の例:**

| 負けた相手コース | 差し | まくり | まくり差し |
|---------------|------|--------|----------|
| 2コース | 30% | 10% | 5% |
| 3コース | 5% | 15% | 10% |
| 4コース | 5% | 10% | 10% |

## 超展開データの詳細

### 定義

超展開データとは、**各選手が1マークでどんな攻め方をして、どれだけ勝っているか、また、1マークでどんな攻められ方をして、どれだけ負けているかがわかるデータ**。2025年9月12日に公開された最新のオリジナルデータ。

従来はYouTubeメンバーシップでのみ公開されていたが、当日レース分はボートレース日和で無料閲覧可能になった。

### 決まり手データとの違い

| 観点 | 決まり手データ | 超展開データ |
|------|-------------|------------|
| 分析対象 | 勝った時の決まり手率 | 1マークでの攻め方・攻められ方の全体像 |
| 1コースの追加情報 | なし | 抵抗回数（まくりに対して抵抗した回数） |
| 2〜6コースの追加情報 | なし | 絞った回数（内側の艇を絞り込んだ回数） |
| 未勝利コースの分析 | 不可能 | 該当コースで1回も勝っていない選手の攻め方も予測可能 |

### 1マークの攻め方の可視化

#### 1コース（インコース）のデータ

| 項目 | 説明 |
|------|------|
| 勝った時の決まり手分布 | 逃げ・差し等の割合 |
| 負けた時の攻められ方 | どのコースからどの決まり手で負けたか |
| 抵抗回数 | まくってきた外枠艇に対して抵抗（ブロック）した回数 |

#### 2〜6コース（アウトコース）のデータ

| 項目 | 説明 |
|------|------|
| 勝った時の決まり手分布 | 差し・まくり・まくり差し等の割合 |
| 負けた時の攻められ方 | 他コースからどう負けたか |
| 絞った回数 | 内側の艇を絞り込んだ（外から圧をかけた）回数 |

### アクセス方法

1. トップ画面で場を選択
2. 出走一覧から「枠別勝率」を選択
3. 下にスクロール
4. 「超展開データ」ボタンをクリック

### 閲覧制限

| 閲覧範囲 | 対象 |
|---------|------|
| 当日レースの超展開データ | 無料（ボートレース日和上） |
| 全日の超展開データ + 選手詳細 + レース検索 | 有料（YouTubeメンバーシップ） |

### 判定基準ガイドライン

超展開データには判定基準を示すガイドライン表が用意されており、各数値の大小を評価する際の基準が提供されている（具体的な閾値は非公開）。

## オリジナル展示タイム

### 概要

2024年9月に大幅拡充されたオリジナル展示タイム分析機能。公式の展示タイムに加え、独自に計測・分析したタイムデータを提供する。

### 提供データ一覧

#### 選手別周回タイム分析

| 項目 | 説明 |
|------|------|
| 周回タイム | 選手別の1周あたりのタイム |
| 周足タイム | 足回り（ターンを含む）のタイム |
| 直線タイム | 直線区間のタイム |

※オリジナル展示タイムが公開されている場のみ計測可能。

#### 周回タイム比較

同一レースの出場選手間で周回タイムを比較する機能。

#### 展示タイム比較

展示タイムを選手間で比較し、どの選手が突出して速いかを可視化。まくりアラート機能と連動している。

#### 場別オリ展1位1着率一覧表

各場ごとに「オリジナル展示タイム1位の選手が実際に1着になった率」を一覧表示。場によって展示タイムの信頼度が異なることを数値で確認できる。

#### 直前情報での追加表示項目

2024年以降、直前情報タブに以下の項目が追加された。

| 項目 | 説明 |
|------|------|
| 前走展示タイム | 前のレースでの展示タイム |
| 前走周回タイム | 前のレースでの周回タイム |
| 前走周足タイム | 前のレースでの周足タイム |
| 前走直線タイム | 前のレースでの直線タイム |
| 今節平均周回 | 今節の平均周回タイム |
| 今節平均周足 | 今節の平均周足タイム |
| 今節平均直線 | 今節の平均直線タイム |

### 展示ST vs 本番STのズレ評価

2025年8月に追加された機能。展示のスタートタイミング（ST）と本番のSTがどれだけ一致するかを評価する。

#### 評価ランク

| ランク | 意味 |
|--------|------|
| SS | 展示と本番でほぼ同じSTで安定している（最高評価） |
| S | ズレが非常に小さい |
| A | ズレが小さい |
| B | ズレがやや大きい |
| C | ズレが大きい（展示STは参考にならない） |

**活用方法**: SSやSランクの選手は展示STを信頼してスタート予想が可能。Cランクの選手は展示STを鵜呑みにせず、平均STや今節STを重視すべき。

## データの活用方法

### 1マーク展開予想の基本フロー

```
1. 各選手の枠別決まり手データを確認（新概念データ含む）
   ↓
2. 1号艇の逃げ率 + 2号艇の逃がし率を合算
   → 合計が100%超 → イン逃げが濃厚
   → 合計が100%未満 → 外枠からの攻めを警戒
   ↓
3. 超展開データで1マークの攻め方を確認
   → 絞り回数が多い選手がいれば展開が荒れる可能性
   → 1コースの抵抗回数が多ければインの粘り強さを評価
   ↓
4. グレード別に切り替えて傾向の一貫性を確認
   ↓
5. 展示タイムとオリ展データで機力差を評価
   → まくりアラートと合わせて外からの攻めを判断
```

### イン逃げ信頼度の判断基準

| 条件 | 判断 |
|------|------|
| 1号艇逃げ率 + 2号艇逃がし率 > 100% かつ 外枠決まり手率 < 10% | イン逃げ信頼度：高 |
| 1号艇逃げ率 + 2号艇逃がし率 > 100% だが外枠の攻めあり | イン逃げ信頼度：中 |
| 1号艇逃げ率 + 2号艇逃がし率 < 100% | イン逃げ信頼度：低 |

### 穴狙いの判断基準

| 条件 | 判断 |
|------|------|
| 1号艇のまくられ率が高い | 外枠まくりの可能性 |
| 1号艇の差され率が高い | 2・3コース差しの可能性 |
| 超展開データで外枠の絞り回数が多い | 積極的に攻めるスタイル |
| 展示タイムが外枠のほうが速い | 機力差でまくり成立の可能性 |

## BoatAIへの実装示唆

### 短期的に実装可能なもの

| 機能 | 実装案 | 必要データ |
|------|--------|-----------|
| 枠別決まり手率の表示 | 出走表にコース別の逃げ/差し/まくり/まくり差し率を追加 | 過去レース結果の決まり手データ |
| イン逃げ信頼度スコア | 1号艇逃げ率 + 2号艇逃がし率の合算値を算出 | 同上 |
| グレード別データ切り替え | 総合/一般戦/SG・G1/女子戦のフィルタリング | レースのグレード情報 |

### 中期的に実装を検討するもの

| 機能 | 実装案 | 課題 |
|------|--------|------|
| 新概念データ（負け方分析） | 1コースが負けた時の決まり手内訳を計算 | 過去データの決まり手を「勝者」と「敗者」両方で集計する必要あり |
| 展示ST vs 本番STのズレ評価 | 過去の展示ST・本番STを蓄積し、選手別にズレの標準偏差を計算 | 展示STデータのスクレイピングが必要 |

### 長期的な差別化ポイント

| 機能 | BoatAIならではのアプローチ |
|------|-------------------------|
| 超展開データ相当の分析 | ルールベースの予測モデルに1マーク展開予測を組み込み、決まり手の確率分布を予測に反映 |
| 展開予測の自動化 | 決まり手データ + 新概念データ + 展示タイム + ST傾向を統合し、1マーク展開をシミュレーション |
| 回収率への直結 | ボートレース日和は「データを見せる」までだが、BoatAIは「データに基づいて予測を出す」ところまで踏み込める |

### データ収集の優先順位

1. **決まり手データ**: 公式サイトのレース結果から取得可能（既存スクレイピングの拡張）
2. **負け方データ**: レース結果の決まり手 + 着順から逆算可能
3. **展示タイム**: 直前情報のスクレイピングで取得（タイミングに制約あり）
4. **展示ST**: 直前情報からの取得が必要（本番STとの比較用に蓄積）
