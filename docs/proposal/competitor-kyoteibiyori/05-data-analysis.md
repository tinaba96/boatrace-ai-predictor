# データ分析機能の詳細分析

> **対象サイト**: ボートレース日和（kyoteibiyori.com）
> **対象URL**: https://kyoteibiyori.com/data/

## 概要

ボートレース日和の「データ分析」は、過去レースデータを多角的に検索・分析できる統合ツール群。トップページ上部の「データ分析」メニューからアクセスし、タブ切り替えで9つの主要サブ機能（+ フライング休み期間、売上履歴）を利用できる。

すべての機能は無料で利用可能。一部機能では検索条件を名前付きで保存し、繰り返し利用できる「条件登録」機能がある。

### サブ機能一覧

| # | サブ機能 | 主な用途 |
|---|---------|---------|
| 1 | 決まり手検索 | 決まり手条件で過去レースを検索 |
| 2 | 連対率検索 | 本日/明日のレースを連対率条件で検索 |
| 3 | レース分析 | 日別/月別の着順条件検索 |
| 4 | 番組解析 | 年度・グレード別の番組傾向分析 |
| 5 | 勝率・枠別ランキング | 選手のコース別成績ランキング |
| 6 | 高配当オッズランキング | 高額配当レースの一覧 |
| 7 | 優勝履歴 | 年度別優勝者検索 |
| 8 | SG履歴 | SGレース別の歴代成績 |
| 9 | 平均値データ | 会場・級別の各種統計平均値 |

---

## 各サブ機能の詳細

### 1. 決まり手検索

過去レースを「決まり手」を中心とした条件で絞り込み検索する機能。以前は当日22時以降のレースのみ検索可能だったが、現在は過去レースの検索にも対応。

#### 検索条件

| 条件 | 選択肢 | 備考 |
|------|--------|------|
| 期間 | 直近6ヶ月 / 直近1年 | 2段階 |
| 日付指定 | 日付ピッカー | 特定日の指定も可能 |
| 場 | 全24場から選択 | 桐生〜大村 |
| 1〜6号艇 | 各艇に個別フィルター | 決まり手条件を艇番ごとに指定 |
| その他 | 上記以外の条件 | 補助条件 |
| ST順位 | ST順位の指定 | ST順位の差も指定可能 |

#### 艇番ごとの決まり手条件

各号艇（1〜6号艇）に対して以下を個別に指定できる:

- 逃げ率（1号艇の場合）
- 差し率
- 捲り率
- 捲り差し率
- 逃し2着率（1号艇が逃げ損ねた場合の2着率）

#### 出力データ

- 条件に合致するレース一覧
- 各レースの会場、日時、レース番号
- 各艇の着順結果
- 決まり手情報
- 結果は時系列順にソート可能

#### 追加機能

- **条件登録**: 検索条件を名前付きで保存可能
- **条件一覧**: 保存済み条件の一覧表示・呼び出し
- **条件リセット**: 全条件の初期化
- **明日のレース検索**: 翌日のレース番組に対して条件検索

#### 活用シーン

- 「1号艇の逃げ率が低く、3号艇の捲り率が高いレース」を検索して展開予想の参考にする
- 特定の決まり手パターンが出やすい条件を発見する
- 穴レースの傾向を過去データから分析する

### 2. 連対率検索

本日（19時以降は翌日）のレースに出場する選手の連対率を条件として、注目レースを絞り込む機能。

#### 検索条件

| 条件 | 選択肢 | 備考 |
|------|--------|------|
| 期間 | 直近6ヶ月 / 直近1年 | 統計集計期間 |
| 1〜6号艇 | 各艇に個別フィルター | 連対率条件を艇番ごとに指定 |
| ST順位 | ST順位指定 | ST順位の差も指定可能 |

#### 艇番ごとの連対率条件

各号艇に対して以下を個別に指定できる:

- 1着率（XX%以上/以下）
- 2連対率（XX%以上/以下）
- 3連対率（XX%以上/以下）
- 平均ST
- ST順位

#### 出力データ

- 条件に合致する本日のレース一覧
- 各レースの会場、レース番号
- 出場選手の連対率データ
- 直接レース詳細へのリンク

#### 追加機能

- **条件リセット**: 全条件の初期化
- **明日のレース検索**: 翌日のレースに対して検索

#### 活用シーン

- 「1号艇の1着率が70%以上で、かつ他艇の連対率が低いレース」= 堅い本命レースの発見
- 「外枠に連対率の高い選手がいるレース」= 高配当レースの候補発見
- 特定のコースの連対率パターンから展開を予測

### 3. レース分析

過去レースを日別/月別で、各艇の着順条件と万舟フィルターを使って検索する機能。

#### 検索条件

| 条件 | 選択肢 | 備考 |
|------|--------|------|
| 検索タイプ | 日別検索 / 月別検索 | 日付入力 or 月入力 |
| 場 | 全24場から選択 | 桐生〜大村 |
| 1号艇順位 | すべて / 1着 / 2着 / 3着 / 2着以内 / 3着以内 | 6択 |
| 2号艇順位 | 同上 | 同上 |
| 3号艇順位 | 同上 | 同上 |
| 4号艇順位 | 同上 | 同上 |
| 5号艇順位 | 同上 | 同上 |
| 6号艇順位 | 同上 | 同上 |
| 万舟 | すべて / 万舟のみ | 配当10,000円以上 |

#### 出力データ

- 条件に合致するレース一覧
- 日付、会場、レース番号
- 各艇の着順結果
- 配当金額
- 決まり手

#### 活用シーン

- 「1号艇が1着で、4号艇が2着のレース」のパターン検索
- 万舟（高配当）レースのみを抽出して傾向分析
- 特定会場の月別パターン変化を確認
- 「5号艇が1着になったレース」の共通条件探索

### 4. 番組解析

年度・グレード別に各会場の番組編成傾向を分析する機能。レースが荒れやすいか堅いかを把握するのに有効。

#### 検索条件

| 条件 | 選択肢 | 備考 |
|------|--------|------|
| 年度 | 2019年度〜2025年度 | 7年度分 |
| グレード | 一般戦・G3 / SG・G1・G2 / すべて | 3択 |
| 場 | 全24場から選択 | 桐生〜大村 |

#### 出力データ

番組解析では以下のデータが表示される:

- **平均払戻**: 各レース番号ごとの平均配当金額
- **万舟率**: 配当10,000円以上のレースの出現割合
- **1コース1着率**: 1号艇が逃げ切る確率
- **決まり手比率**: 逃げ/差し/捲り/捲り差し/抜き/恵まれの比率
- **レース番号別傾向**: 1R〜12Rごとの傾向差

#### 活用シーン

- 「大村の一般戦は1Rの万舟率が低く、12Rは荒れやすい」といった傾向把握
- 年度間の比較で会場の傾向変化を検出
- グレード別（一般戦 vs SG）での番組傾向の違いを分析
- 平均払戻と万舟率を見てそのレースが荒れやすいか順当かを判断

| 7 | 捲り | 捲り決まり手率 |
| 8 | 捲差 | 捲り差し決まり手率 |

**グループ2（枠別/コース別）:**

| # | 勝率種類 | 内容 |
|---|---------|------|
| 1 | 1着率 | コース別1着率 |

フライング（スタート違反）による出場停止期間の情報を表示。検索フォームは不要で、情報一覧が表示される。

## 共通UI要素

すべてのサブ機能に共通する操作:

- **検索ボタン**: 条件を設定後に実行
- **条件リセット**: すべての条件を初期値に戻す
- **タブ切り替え**: サブ機能間の移動

## BoatAIへの実装示唆

### 高優先度

| 機能 | 理由 | 実装方針 |
|------|------|---------|
| 番組解析 | 会場別・レース番号別の荒れやすさを把握でき、ルール設計に直結 | `data/analysis/` に会場別の番組傾向データを蓄積 |
| 勝率・枠別ランキング | 選手の会場×コース別成績はルールマッチングの精度向上に寄与 | `ruleMatchService.js` での条件判定に選手ランキングデータを活用 |
| レース分析（万舟フィルター） | 万舟レースの共通パターン抽出により、高配当ルールの精度向上 | 分析スクリプトに万舟条件のフィルターを追加 |

### 中優先度

| 機能 | 理由 | 実装方針 |
|------|------|---------|
| 決まり手検索 | 展開予想の精度向上。特に外枠の差し/捲り率は重要 | 決まり手データをルール条件に組み込み |
| 高配当オッズランキング | ユーザーのエンゲージメント向上コンテンツとして | ブログ記事での月次レポートに高配当ランキングを追加 |
| 連対率検索 | 本日のレースのリアルタイム分析に有用 | 予測生成時に連対率条件フィルターを追加 |

### 低優先度

| 機能 | 理由 | 実装方針 |
|------|------|---------|
| 優勝履歴/SG履歴 | 直接的な予測精度への影響は小さい | コンテンツ充実のための参考データ |
| 平均値データ | 会場分析の基礎データとして既に `data/analysis/` で管理 | 既存の分析フローの補完 |
| 売上履歴 | データソースの確保が課題 | 将来的な検討事項 |

### BoatAIの差別化ポイント

ボートレース日和のデータ分析は「検索・閲覧」に特化しているが、BoatAIは**分析結果を予測に直結させる**点で差別化できる:

1. **ルールベース自動適用**: 手動で検索条件を設定する必要なく、過去データから発見したルールを自動で予測に反映
2. **回収率ベースの評価**: 的中率だけでなく回収率100%超えを基準とした厳選型アプローチ
3. **会場特化モデル**: 15会場34ルールの蓄積を活かした、会場ごとの精密な予測

ボートレース日和が「データの百科事典」なら、BoatAIは「データを活用した予測エンジン」というポジションを取れる。
