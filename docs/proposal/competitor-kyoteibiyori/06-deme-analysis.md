# 出目分析機能の詳細分析

> **対象サイト**: ボートレース日和（kyoteibiyori.com）
> **対象URL**: https://kyoteibiyori.com/tools/tools.php

## 概要

「出目分析」は、過去レースの着順パターン（出目）を期間・会場・コース・決まり手・勝式で絞り込み、統計的に分析する機能。トップメニューの「出目分析」からアクセスできる。

ボートレースでは「出目」が会場ごとに明確な傾向を持つため、会場特性の把握や舟券戦略の構築に広く活用されている。ボートレース日和の出目分析は、公式サイトの出目データよりも細かい条件指定が可能で、決まり手と組み合わせた分析ができる点が特徴。

### 出目分析の2つの側面

ボートレース日和では出目データを2つの観点から提供している:

| 種類 | 場所 | 内容 |
|------|------|------|
| 出目分析ツール | /tools/tools.php | 期間・条件を指定して出目統計を検索（本ドキュメントの対象） |
| 出目ランク（出走表内） | 各レースの出走表 | 選手枠別/レース別の出目ランキングをレースごとに表示 |

**選手枠別出目ランキング**: 出場選手6名がその艇番で戦った過去データから、出やすい出目を算出。選手の実力と枠番の組み合わせに基づく個別分析。

**レース別出目ランキング**: その会場・そのレース番号で過去に出やすかった出目を表示。番組編成の傾向に基づく統計分析。

## 検索条件の全詳細

### 1. 期間指定

| 項目 | 仕様 |
|------|------|
| 入力形式 | 日付ピッカー（From / To） |
| From | 集計開始日 |
| To | 集計終了日 |
| デフォルト | 不明（おそらく直近数ヶ月） |
| 最大範囲 | 制限の詳細は不明だが、年単位の検索が可能 |

任意の期間を指定できるため、「直近1ヶ月」「直近半年」「直近1年」など、分析目的に応じた柔軟な期間設定が可能。季節変動の分析にも対応できる。

| 選択肢 | 全25択（すべて + 24場） |

選択可能な全会場:

「すべて」を選択すると全24場の合算データが表示される。

### 3. 設定条件の切り替え

| 項目 | 仕様 |
|------|------|
| 入力形式 | 切り替えボタン（2択） |
| 選択肢 | コース・決まり手 / 決り手 |

- **コース・決まり手**: 1着コース指定と決まり手指定の両方を組み合わせて検索
- **決り手**: 決まり手のみで検索（コース指定なし）

### 4. 1着コース指定

| 項目 | 仕様 |
|------|------|
| 入力形式 | ラジオボタン（7択） |
| デフォルト | 全コース |

選択肢:

| # | 選択肢 | 説明 |
|---|--------|------|
| 1 | 全コース | コース制限なし（すべての1着コースを含む） |
| 2 | 1コース | 1コース（最内）からの1着のみ |
| 3 | 2コース | 2コースからの1着のみ |
| 4 | 3コース | 3コースからの1着のみ |
| 5 | 4コース | 4コースからの1着のみ |
| 6 | 5コース | 5コースからの1着のみ |
| 7 | 6コース | 6コース（最外）からの1着のみ |

注意: ここでの「コース」は枠番（艇番）ではなく、実際の進入コースを指す。進入変化がなかったレースのみが集計対象となるため、枠番=コースとなる。

### 5. 決まり手指定

| 項目 | 仕様 |
|------|------|
| 入力形式 | ラジオボタン（4択） |
| デフォルト | 全 |

選択肢:

| # | 選択肢 | 説明 |
|---|--------|------|
| 1 | 全 | 決まり手制限なし |
| 2 | 差し | 内側から差して1着になったレース |
| 3 | 捲り | 外側から捲って1着になったレース |
| 4 | 捲り差し | 外側から捲りながら差して1着になったレース |

注意: 「逃げ」「抜き」「恵まれ」は選択肢に含まれていない。これは出目パターンとして分析する際、「逃げ」は1コース1着がほぼ確定で出目のバリエーションが限られ、「抜き」「恵まれ」はレアケースで統計的意味が薄いためと考えられる。

### 6. 勝式選択

| 項目 | 仕様 |
|------|------|
| 入力形式 | ラジオボタン（3択） |
| デフォルト | 単勝 |

選択肢:

| # | 勝式 | 組み合わせ数 | 説明 |
|---|------|------------|------|
| 1 | 単勝 | 6通り | 1着の艇番のみ |
| 2 | 2連単 | 30通り | 1着-2着を着順通り |
| 3 | 3連単 | 120通り | 1着-2着-3着を着順通り |

2連複、3連複、拡連複は選択できない。着順が確定する「単」系の勝式に限定されている。

### 7. 進入固定オプション

| 項目 | 仕様 |
|------|------|
| 適用方法 | 自動適用（常にON） |
| 条件 | 進入変化しなかったレースのみを集計 |

ページ上に「※進入変化しなかったレースのみを集計」と注記されている。これは手動でON/OFFを切り替えるオプションではなく、出目分析のすべての検索結果に自動的に適用される条件。

理由: 進入変化（枠番とコースが異なる状態）が発生すると、出目の意味が変わるため。例えば「1-2-3」という出目でも、1号艇が実際には3コースから走っていた場合、純粋なコース別傾向分析にならない。

## 出力データの形式

### 結果表示の構造

検索結果は「検索結果」セクションに表示される。表示形式は選択した勝式によって異なる。

#### 単勝の場合

各艇番（1号艇〜6号艇）ごとの1着回数と比率を表示:

| 表示項目 | 内容 |
|---------|------|
| 艇番 | 1号艇〜6号艇 |
| 出現回数 | 該当艇番が1着になった回数 |
| 出現率（割合） | 全レース中の比率 |
| その他 | 上記以外の結果（事故レース等） |

#### 2連単の場合

1着-2着の組み合わせ（30通り）を出現頻度順に表示:

| 表示項目 | 内容 |
|---------|------|
| 出目 | X-Y（1着艇番-2着艇番） |
| 出現回数 | 該当出目の出現回数 |
| 出現率 | 全レース中の比率 |

#### 3連単の場合

1着-2着-3着の組み合わせ（120通り）を出現頻度順に表示:

| 表示項目 | 内容 |
|---------|------|
| 出目 | X-Y-Z（1着-2着-3着） |
| 出現回数 | 該当出目の出現回数 |
| 出現率 | 全レース中の比率 |

### ブログ記事で公開されたデータの形式（参考）

ボートレース日和のブログ記事「全国24競艇場 3連単の出目 的中率、回収率ランキング」では、以下の項目でランキングが作成されている:

| カラム | 内容 |
|--------|------|
| 順位 | ランキング順位 |
| 会場名 | ボートレース場 |
| 3連単出目 | X-Y-Z 形式 |
| 的中率 | 該当出目が出現する確率（例: 9.56%） |
| 回収率 | 該当出目を毎回購入した場合の回収率（例: 89.64%） |
| 対象レース数 | 集計期間内のレース数 |

的中率上位は「1-2-3」「1-2-4」「1-3-2」など1号艇1着のパターンが占め、回収率上位は「6-5-2」のような低出現率・高配当のパターンが占める傾向がある。

## 活用方法

### 1. 会場特性の把握

会場ごとにイン逃げ率や出目傾向は大きく異なる。出目分析で以下を確認:

- **イン有利な会場**: 大村、徳山 → 「1-X-X」パターンの出現率が高い
- **荒れやすい会場**: 江戸川、戸田 → 外枠1着の出目が相対的に多い
- **決まり手の偏り**: 平和島は差しが多い、常滑は捲りが多い等

### 2. 季節・期間別の傾向変化

期間指定を変えることで、季節による出目変化を検出:

- **冬季**: モーターの調整難易度が上がり、荒れやすい
- **夏季**: 水面が安定し、イン逃げ率が上がる傾向
- **期初**: 新モーター導入直後は荒れやすい

### 3. 決まり手別の出目パターン

決まり手を絞ることで、展開別の着順パターンを分析:

- **差し**: 「2-1-X」「2-3-X」など内側からの差しパターン
- **捲り**: 「3-X-X」「4-X-X」など外枠の捲りパターン
- **捲り差し**: 中間枠からの攻めパターン

### 4. 舟券戦略への活用

- **出目買い**: 出現率の高い出目を中心に購入
- **穴狙い**: 出現率は低いが回収率が高い出目を狙う
- **フォーメーション構築**: 出目分析の結果から、買い目のフォーメーションを設計
- **資金配分**: 本命と抑えの資金配分の参考

## 出走表内の出目ランク（関連機能）

出目分析ツールとは別に、各レースの出走表にも「出目ランク」タブが存在する。

### 選手枠別出目ランキング

| 項目 | 内容 |
|------|------|
| 対象 | 出場選手6名のその艇番での過去レース |
| 期間 | 直近1年 |
| 表示 | 出やすい出目をランキング形式 |
| フィルター | 1着艇選択で絞り込み可能 |

選手の個別データに基づくため、同じ会場・レース番号でも出場選手が変われば結果が変わる。選手の実力と枠番の適性を反映した出目予測。

### レース別出目ランキング

| 項目 | 内容 |
|------|------|
| 対象 | その会場・そのレース番号の過去レース |
| 期間 | 直近1年 |
| 表示 | 出やすい出目をランキング形式 |

番組編成の傾向（例: 1Rはイン逃げが多い、12Rは実力が拮抗して荒れやすい）を反映した統計データ。

## BoatAIへの実装示唆

### 直接的に活用可能なデータ

| データ | 活用先 | 実装方法 |
|--------|--------|---------|
| 会場別の出目分布 | 会場分析の基礎データ | `scripts/analysis/` で会場別出目分析スクリプトを作成 |
| コース別×決まり手別の出目 | ルール条件の精緻化 | `ruleMatchService.js` の条件に出目傾向を追加 |
| 進入固定レースの統計 | 予測精度の向上 | 進入変化の有無を考慮した予測モデル |

### 実装優先度

**高優先度: 会場別出目傾向データの蓄積**

BoatAIの既存の会場分析（`data/analysis/venue-XX/`）に出目傾向データを追加することで、ルール設計の根拠が強化される。特に:

- 会場ごとの3連単出目TOP10（的中率・回収率）
- コース別1着率の時系列変化
- 決まり手別の2着・3着パターン

**中優先度: 出目ベースの予測ルール**

出目統計を予測ロジックに組み込む:

- 特定会場×特定レース番号で回収率が100%を超える出目パターンをルール化
- 例: 「大村の1Rで1-2-3は的中率9.56%、回収率89.64%」→ 条件付きで採用検討

**低優先度: ユーザー向け出目表示**

フロントエンドでの出目ランキング表示:

- 各レースの出走表に出目ランキングを追加（ボートレース日和の出目ランクと同等）
- ユーザーが会場別の出目傾向を閲覧できるページの追加

### BoatAIの差別化ポイント

ボートレース日和の出目分析は「過去の出現頻度」のみを表示するが、BoatAIでは:

1. **回収率ベースの出目評価**: 出現率だけでなく、回収率100%超えの出目パターンを特定
2. **条件付き出目分析**: 選手の級別、モーター性能、天候条件を組み合わせた高次元の出目分析
3. **ルール自動適用**: 発見した出目パターンを予測エンジンに自動組み込み
4. **期間×会場のクロス分析**: 複数会場・複数期間の横断的な出目比較

ボートレース日和が「出目の辞書」なら、BoatAIは「出目を活用した予測と資金配分の最適化ツール」として差別化できる。
