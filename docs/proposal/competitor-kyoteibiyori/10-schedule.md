# 開催予定機能の詳細分析

## 概要

ボートレース日和の「開催予定」は、今後のボートレース開催スケジュールを一覧表示する機能。場・グレード・時間帯の3軸でフィルタリングでき、SGやG1といった注目レースを素早く見つけられる設計になっている。

URL: `https://kyoteibiyori.com/schedule/`

## 表示データ

各開催について以下の情報が表示される（推定含む）:

| 項目 | 説明 |
|------|------|
| 開催場名 | ボートレース場の名称 |
| 開催名 | レースタイトル（例: "○○記念"） |
| グレード | SG, G1, G2, G3, 一般 |
| 開催期間 | 開始日〜最終日 |
| 時間帯区分 | モーニング/デイ/ナイター |
| 開催ステータス | 開催中/開催予定 |

## カレンダー表示

- 開催予定はリスト形式で表示
- フィルター条件を適用した結果が検索結果セクションに表示される
- 月単位のカレンダービューは確認できなかった（リスト表示が主）
- 各開催から出走表ページへの直接リンクあり

## BoatAIへの実装示唆

### 優先度: 高

1. **開催カレンダーページ**: SGやG1の開催予定をカレンダー形式で表示。BoatAIの予想対象レースの全体像を把握できるページとして有用。現在のBoatAIにはスケジュール表示機能がないため、新規ページとして追加する価値が高い

2. **フィルター3軸（場/グレード/時間帯）**: ボートレース日和と同等のフィルター機能は最低限必要。BoatAIではさらに「予想配信対象かどうか」のフラグを追加すると差別化できる

### 優先度: 中

3. **SG/G1カウントダウン**: 次のSG開催までの日数カウントダウンを表示し、ユーザーの期待感を醸成する

4. **開催情報と連動した予想ページ**: 開催予定から直接予想ページに遷移できる導線。ボートレース日和は「データ→出走表」の流れだが、BoatAIは「スケジュール→AI予想」という独自の導線を作れる

### 優先度: 低

5. **月間カレンダービュー**: 月単位でどの場がいつ開催しているかを俯瞰できるビジュアルカレンダー。ボートレース日和にはない機能として差別化可能

### 差別化ポイント

- ボートレース日和のスケジュール機能はシンプルなフィルター+リスト表示。BoatAIでは「このSGレースでAI予想を配信予定」といった独自の付加価値を持たせることで差別化を図れる
- フィルターにG2/G3/一般戦の個別選択を加えると、より柔軟な検索が可能になる
