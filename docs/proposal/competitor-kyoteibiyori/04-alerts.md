# アラート機能の詳細分析

## 概要

ボートレース日和（kyoteibiyori.com）のトップページに配置されている5種類のアラート機能。特定条件に合致するレースを当日・翌日のレースから自動検出し、効率的なレース選定を支援する。

すべて無料で利用可能。会員登録は不要だが、登録すると検索条件のカスタマイズやMyDataとの連携が可能になる。

### アラート一覧

| アラート名 | 目的 | 検索タイミング |
|-----------|------|-------------|
| ガチガチレース検索 | 堅い決着が見込まれるレースを発見 | 出走表確定後 |
| 穴レース検索 | 荒れそうなレースを発見 | 出走表確定後 |
| まくりアラート | 展示タイムが突出して速い選手がいるレースを発見 | 展示終了後 |
| 前づけアラート | 展示航走で進入コースが変化したレースを発見 | 展示終了後 |
| チルト跳アラート | チルト角度を大きく上げた選手がいるレースを発見 | 展示終了後 |

---

## 各アラートの詳細

### ガチガチレース検索

#### 概要

1号艇の逃げ率と2号艇の逃し率を条件に指定し、堅い決着（イン逃げ）が想定されるレースを検索する機能。的中重視でリスクを抑えたい場合に使う。

#### 検索条件（パラメータの全選択肢）

**パラメータ1: 1号艇の逃げ率**

| 選択肢 | 意味 |
|--------|------|
| 75%以上 | 非常に高い逃げ率（最も堅い条件） |
| 70%以上 | 高い逃げ率 |
| 65%以上 | やや高い逃げ率 |
| 60%以上 | 標準的に高い逃げ率 |
| 55%以上 | やや甘い条件 |
| 50%以上 | 最も広い条件 |

**パラメータ2: 2号艇の逃し率（逃がし率）**

| 選択肢 | 意味 |
|--------|------|
| 35%以上 | 最も広い条件（2号艇が35%以上の確率で1号艇を逃がす） |
| 40%以上 | やや緩い条件 |
| 45%以上 | 標準的な条件 |
| 50%以上 | やや厳しい条件 |
| 55%以上 | 厳しい条件 |
| 60%以上 | 非常に厳しい条件（最も堅い条件） |

※「逃し率」= 2号艇がイン逃げを許した割合。数値が高いほど2号艇が1号艇の逃げを阻止しない傾向。

#### 検索ロジック

```
1号艇の逃げ率 >= 指定値 AND 2号艇の逃し率 >= 指定値
→ 条件に合致するレースを一覧表示
```

#### 出力形式

- 条件に合致したレースを会場・レース番号ごとに一覧表示
- 会場をタップすると各レースの簡易出走表が表示される
- さらに詳細を見たい場合、基本情報・枠別情報・今節成績・直前情報・オッズ検索・オッズ一覧・出目ランク・MyDataの各タブに遷移可能

#### 使い方

1. 1号艇の逃げ率を選択（推奨: 65%以上〜75%以上）
2. 2号艇の逃し率を選択（推奨: 50%以上〜60%以上）
3. 「本日のレース検索」または「明日のレース検索」をタップ
4. 該当レースが一覧表示される
5. 気になるレースをタップして詳細データを確認

**運用のコツ**: 大きな配当は狙えないため、単勝・複勝・2連複など低リスクの舟券に適している。コツコツ積み上げる戦略向け。

### 穴レース検索

#### 概要

1号艇の逃げ率が低く、かつ外枠の差し・まくり率が高いレースを検索する機能。荒れるレースを見つけて高配当を狙う場合に使う。

#### 検索条件（パラメータの全選択肢）

**パラメータ1: 1号艇の逃げ率**

| 選択肢 | 意味 |
|--------|------|
| 50%以下 | 最も広い条件 |
| 40%以下 | やや絞った条件 |
| 30%以下 | 逃げ率がかなり低い |
| 20%以下 | 非常に低い逃げ率 |
| 10%以下 | 極端に低い逃げ率（最も荒れやすい条件） |

**パラメータ2: 2〜6号艇の差し・捲り率**

| 選択肢 | 意味 |
|--------|------|
| 20%以上 | 最も広い条件 |
| 25%以上 | やや絞った条件 |
| 30%以上 | 標準的な攻撃力 |
| 35%以上 | 高い攻撃力 |
| 40%以上 | 非常に高い攻撃力（最も厳しい条件） |

※差し・捲り率は2〜6号艇のうち最も高い選手の値で判定される。

#### 検索ロジック

```
1号艇の逃げ率 <= 指定値 AND MAX(2〜6号艇の差し・捲り率) >= 指定値
→ 条件に合致するレースを一覧表示
```

#### 出力形式

ガチガチレース検索と同様の形式。ただし、差し・捲り率が最も高い艇がハイライト表示される。

#### 使い方

1. 1号艇の逃げ率を選択（推奨: 30%以下〜40%以下）
2. 差し・捲り率を選択（推奨: 30%以上〜35%以上）
3. 「本日のレース検索」または「明日のレース検索」をタップ
4. 該当レースが一覧表示される
5. 差し・捲り率が一番高い艇を軸に、他の入線候補を自分で選定

**運用のコツ**: 差し・捲り率が高い艇は1着候補だが、2着以降は自力で分析が必要。3連単や3連複で高配当を狙う戦略向け。

### まくりアラート

#### 概要

展示タイムが他の艇より突出して速い選手がいるレースを検索する機能。展示タイムが速いということは直線でのスピードの伸びを重視した「伸び型調整」をしている可能性が高く、まくりでの決着が期待できる。

#### 検索条件（パラメータの全選択肢）

**パラメータ1: 展示タイム差**

| 選択肢 | 意味 |
|--------|------|
| 0.07以上 | 最も広い条件（0.07秒以上速い選手がいるレース） |
| 0.08以上 | やや絞った条件 |
| 0.09以上 | 絞った条件 |
| 0.10以上 | 最も厳しい条件（0.10秒以上のタイム差） |

**パラメータ2: 比較対象**

| 選択肢 | 意味 |
|--------|------|
| 内側艇 | その選手より内側のコースの艇と比較（まくりの直接対象） |
| 全艇 | レース全艇と比較（全体的な機力差を評価） |

#### 検索ロジック

```
いずれかの艇の展示タイムが、比較対象の全艇より指定秒以上速い
→ 条件に合致するレースを一覧表示
```

#### 出力形式

- 該当レースの一覧
- 展示タイムが突出して速い選手がハイライト表示される
- 各レースの展示タイム比較データが確認可能

#### 使い方

1. 展示タイム差を選択（推奨: 0.08以上〜0.09以上）
2. 比較対象を選択（まくり狙いなら「内側艇」、機力差評価なら「全艇」）
3. 「本日のレース検索」をタップ（展示終了後のレースが対象）
4. 該当レースが一覧表示される

**注意事項**: 展示が終了したレースのみが検索対象。検索タイミングに注意が必要。

**運用のコツ**: 展示タイムが速い = 伸び型調整。外枠の選手が内側艇より0.08秒以上速い場合、まくりの可能性が高い。ただしターン技術も考慮する必要がある。

### 前づけアラート

#### 概要

展示航走（スタート展示）で前づけが行われ、進入コースが枠番通りでなくなったレースを検索する機能。進入コース変化はレース展開に大きな影響を与えるため、事前把握が重要。

#### 検索条件

**パラメータ: なし**

前づけアラートには条件設定はない。「本日のレース検索」ボタンをタップするだけで、展示航走で進入変化があったレースがすべて表示される。

#### 用語説明

**前づけ（前付け）** とは、待機行動におけるコース取りの攻防の際に、外枠の艇が大きく回り込み他の艇の前につけ、インコースを取ろうとする戦法のこと。前づけが行われると枠番通りの進入にならず、レース展開が大きく変わる。

#### 出力形式

- 展示航走で進入が変化したレースの一覧
- 変化前（枠番通り）と変化後の進入コースが表示される

#### 使い方

1. 「本日のレース検索」をタップ（展示終了後のレースが対象）
2. 進入変化があったレースが一覧表示される
3. 進入コースの変化を確認し、展開予想を修正

**運用のコツ**: 前づけが行われたレースはインが不利になりやすいため、ガチガチレース検索の結果と照合して「前づけが入っていない堅いレース」を選ぶ使い方が有効。穴狙いの場合は前づけレースに積極的に参加する戦略もある。

---

### チルト跳アラート

#### 概要

展示航走でチルト角度を大きく上げた（跳ねた）選手がいるレースを検索する機能。チルトを上げることは「伸び足重視」の積極的な調整であり、まくり狙いの意思表示ともいえる。

#### 検索条件（パラメータの全選択肢）

**パラメータ: チルト角度**

| 選択肢 | 意味 |
|--------|------|
| +0.5以上 | 最も広い条件（チルトを0.5度以上上げた選手がいるレース） |
| +1以上 | やや絞った条件 |
| +2以上 | 最も厳しい条件（大きくチルトを上げた選手のみ） |

#### 用語説明

**チルト** とは、モーターを艇に取り付ける角度のこと。

| チルト角度 | 効果 |
|-----------|------|
| 下げる（マイナス方向） | 艇が安定し、ターン性能が向上。差しやインコースに有利 |
| 上げる（プラス方向） | 直線での伸びが良くなる。まくりや外コースに有利。ただしターンは不安定に |

通常のチルト角度は-0.5〜0程度。+0.5以上は「跳ねた」とみなされ、選手が伸び足に賭けた積極的な調整をしていることを示す。

#### 出力形式

- チルトを跳ねた選手がいるレースの一覧
- チルトを跳ねた選手の艇番・チルト角度が表示される

#### 使い方

1. チルト角度の条件を選択（推奨: +0.5以上 で広く探索）
2. 「本日のレース検索」または「明日のレース検索」をタップ
3. 該当レースが一覧表示される
4. チルトを跳ねた選手の枠番・コース・展示タイムを確認

**運用のコツ**: チルトを跳ねた選手が外枠にいる場合、まくり狙いの可能性が高い。まくりアラートと合わせて確認すると精度が上がる。ただしチルトを上げるとターンが不安定になるため、選手のまくり実績（超展開データの絞り回数等）も確認すべき。

## 共通仕様

### 本日/明日の切り替え

すべてのアラートに「本日のレース検索」「明日のレース検索」の2つのボタンが配置されている。

| ボタン | 対象 | 注意点 |
|--------|------|--------|
| 本日のレース検索 | 当日開催のレース | まくりアラート・前づけアラート・チルト跳アラートは展示終了後のレースのみ対象 |
| 明日のレース検索 | 翌日開催のレース | 展示情報を使うアラート（まくり・前づけ）は翌日分では使えない場合がある |

### データ更新タイミング

| アラート | 使用データ | 更新タイミング |
|---------|-----------|-------------|
| ガチガチレース検索 | 枠別勝率（逃げ率・逃し率） | 出走表確定時点 |
| 穴レース検索 | 枠別勝率（逃げ率・差し捲り率） | 出走表確定時点 |
| まくりアラート | 展示タイム | 展示航走終了後（レースごとに異なる） |
| 前づけアラート | 展示航走の進入情報 | 展示航走終了後 |
| チルト跳アラート | チルト角度 | 展示航走終了後 |

### 検索結果からの遷移

すべてのアラートの検索結果から、各レースの出走表に遷移可能。出走表では以下のタブを切り替えて詳細分析ができる。

- 基本情報
- 枠別情報（新概念データ・超展開データ含む）
- モータ情報
- 今節成績
- 直前情報
- オッズ検索
- オッズ一覧
- 出目ランク
- MyData

### UIの配置

トップページのメイン領域に5つのアラートがカード形式で縦に並んでいる。各カードには検索条件のプルダウンと検索ボタンが配置されたシンプルなUIで、1タップで検索結果にアクセスできる。

## アラート機能の組み合わせ活用

### パターン1: 堅いレースを確実に獲る

```
ガチガチレース検索（逃げ率70%以上 + 逃し率55%以上）
  → 該当レースを抽出
  → 前づけアラートで進入変化がないことを確認
  → 単勝・複勝・2連複で購入
```

### パターン2: 穴レースで高配当を狙う

```
穴レース検索（逃げ率30%以下 + 差し捲り率30%以上）
  → 該当レースを抽出
  → まくりアラート・チルト跳アラートで外枠の攻め意欲を確認
  → 超展開データで1マークの展開を予測
  → 3連単・3連複で購入
```

### パターン3: まくりレースを狙い撃つ

```
まくりアラート（タイム差0.08以上 + 内側艇比較）
  → 該当レースを抽出
  → チルト跳アラートで同じ選手がチルトを跳ねているか確認
  → 超展開データで当該選手の絞り回数を確認
  → 条件が揃えば高確信度のまくり予想
```

## BoatAIへの実装示唆

### 即時実装可能（既存データで対応）

| 機能 | 実装方法 | 難易度 |
|------|---------|--------|
| ガチガチレース検索相当 | racesテーブルの選手データから枠別逃げ率・逃し率を計算し、条件に合致するレースを抽出 | 低 |
| 穴レース検索相当 | 同上。逃げ率が低く差し捲り率が高いレースを抽出 | 低 |

### 展示データ取得後に実装可能

| 機能 | 実装方法 | 難易度 |
|------|---------|--------|
| まくりアラート相当 | 展示タイムをスクレイピングし、内側艇・全艇との差を計算 | 中 |
| 前づけアラート相当 | 展示航走の進入コースをスクレイピングし、枠番との差異を検出 | 中 |
| チルト跳アラート相当 | チルト角度データをスクレイピングし、条件に合致するレースを抽出 | 中 |

### BoatAIならではの付加価値

| 既存機能 | BoatAI独自の強化案 |
|---------|-------------------|
| ガチガチレース検索 | ルールマッチングと組み合わせ、回収率100%超えの堅いレースのみ表示 |
| 穴レース検索 | 過去の穴レースパターンとのマッチングで、期待回収率を算出 |
| まくりアラート | 超展開データ相当の決まり手分析と統合し、まくり成功確率を算出 |
| 前づけアラート | 前づけ選手の過去の前づけ成功率・失敗率を合わせて表示 |
| チルト跳アラート | チルトを跳ねた選手の過去のまくり成功率と合わせて評価 |

### 実装の優先順位

1. **ガチガチレース検索** - 既存データで即実装可能。BoatAIの堅実予想との相性が良い
2. **穴レース検索** - 同上。高配当狙いのユーザー向け機能
3. **まくりアラート** - 展示タイムの取得が前提だが、差別化効果が高い
4. **チルト跳アラート** - まくりアラートと同時に実装すると効果的
5. **前づけアラート** - 進入情報の取得が必要。実装コストに対して効果は限定的

### UI設計案

ボートレース日和のアラート機能はトップページに配置されてアクセスしやすい。BoatAIでも同様にトップページまたは予測一覧ページにアラート機能を配置し、「今日の注目レース」として以下のセクションを設けることが考えられる。

```
[今日の注目レース]
├── 堅いレース（ガチガチ）: N件
├── 穴レース: N件
├── まくり注意: N件
├── 進入変化あり: N件
└── チルト跳ね: N件
```

各セクションをタップするとレース一覧が表示され、さらにBoatAIの予測データ・ルールマッチ結果と合わせて表示する形が理想的。
