# モーター一覧機能の詳細分析

## 概要

ボートレース日和（kyoteibiyori.com）の「モーター一覧」機能は、全24競艇場のモーター性能データを横断的に閲覧・分析できる機能である。URL: https://kyoteibiyori.com/motor/

競艇場ごとにモーターの成績・使用履歴・部品交換履歴を可視化し、独自指標「貢献ポイント（貢献P）」によりモーター性能を選手の実力と切り分けて評価できる点が最大の特徴。会員登録不要で利用可能。

## 個別モーター詳細

モーター番号をクリックすると、モーダルウィンドウで詳細データが表示される。

### 成績データ

| 項目 | 説明 |
|------|------|
| 通算成績 | モーター抽選日からの全期間の成績 |
| 直近1ヶ月成績 | 直近1ヶ月間の勝率・連率 |
| 勝率 | 通算/直近の勝率 |
| 1着率 | 通算/直近の1着率 |
| 2連率 | 通算/直近の2連対率 |
| 3連率 | 通算/直近の3連対率 |
| 出走数 | 通算/直近の出走回数 |
| 優出回数 | 優勝戦進出回数 |
| 優勝回数 | 優勝回数 |
| 貢献P | 独自の貢献ポイント |

### 使用履歴

モーターを使用した全選手の履歴を時系列で表示。

| 項目 | 説明 |
|------|------|
| 使用選手名 | そのモーターを使用した選手名 |
| 使用前選手勝率 | 選手がモーター使用前の直近6ヶ月の勝率 |
| 使用前選手1着率 | 同上の1着率 |
| 使用前選手2連率 | 同上の2連対率 |
| 使用前選手3連率 | 同上の3連対率 |
| 節の着順 | その節での各レースの着順 |
| グレード | SG / G1 / G2 / G3 / PG1 / 一般戦 |
| 優出の有無 | その節で優勝戦に進出したかどうか |

- 「使用前選手勝率」と実際の節成績を比較することで、モーター性能を推定できる
- 強い選手が使っても成績が悪い = モーター性能が低い可能性
- 弱い選手が使っても成績が良い = モーター性能が高い可能性

### 決まり手データ

モーター使用時の1着における決まり手を集計。

| 決まり手 | 説明 |
|---------|------|
| 逃げ | 1コースからの逃げ切り |
| 差し | インから差しての勝利 |
| 捲り | アウトからのまくり |
| 捲差 | まくり差しでの勝利 |
| 抜き | 2周目以降の抜き |
| 恵まれ | 他艇事故等での繰り上がり |

- 各決まり手の確率（%）を表示
- 円グラフで可視化（詳細は「グラフ・可視化」セクション参照）
- 逃げ率が高いモーターは出足が良い、捲り率が高いモーターは伸び足が良いなどの推測が可能

### 部品交換履歴

モーターの部品交換をした全記録を一覧表示。

| 項目 | 説明 |
|------|------|
| 交換日 | 部品交換を行った日付 |
| 交換部品名 | 交換した部品の名称 |
| 交換選手名 | 部品交換を行った選手名 |

**主な交換部品:**
- ピストン
- ピストンリング
- シリンダーケース
- クランクシャフト
- キャブレター
- 電気一式
- ギアケース
- キャリアボデー

- 部品交換が多いモーターは基本的に「機力がイマイチ」と判断できる
- ただし、整備上手な選手が部品交換後に成績が向上するケースもある
- 2021年5月に部品交換履歴機能が追加
- 2023年4月に部品交換一覧機能として強化

### 中間整備記録

競艇場側が実施する中間整備の記録を表示。

| 項目 | 説明 |
|------|------|
| 整備日 | 中間整備が行われた日付 |
| 整備部品名 | 交換された部品の名称 |
| 備考 | 整備に関する補足情報 |

- 2022年9月に出走表とモーター履歴に中間整備情報を追加
- 中間整備は選手ではなく競艇場の整備士が実施するもの
- 中間整備が入ったモーターは性能が変化する可能性がある
- MyDataにも登録可能

## グラフ・可視化

モーター詳細画面内の「グラフ表示」ボタンから各種グラフにアクセスできる。2022年3月に追加された機能。

### 棒グラフ（バーチャート）

**用途: モーター性能の節ごとの推移を可視化**

表示項目:
- 1着率（節ごと）
- 2着率（節ごと）
- 3着率（節ごと）
- モーター勝率（節ごと）
- 使用前選手勝率（節ごと）

- 各節の使用選手とともに棒グラフで表示
- モーター勝率と使用前選手勝率を並べて表示することで、モーターが選手の実力を上回る成績を出しているか（=モーター性能が良い）、下回っているか（=モーター性能が悪い）を視覚的に判断できる

### 円グラフ

**用途: 決まり手の割合を可視化**

- 逃げ、差し、捲り、捲差、抜き、恵まれの6種類の決まり手率を円グラフで表示
- モーターの得意パターンを一目で把握可能
- 逃げが多い = 出足・行き足が良いモーター
- 捲りが多い = 伸び足が良いモーター
- 差しが多い = 回り足が良いモーター

### 折れ線グラフ（推定）

**用途: 勝率・連対率の時系列推移**

- 節ごとの連対率、勝率の推移を折れ線で表示
- 選手の使用前勝率との差を時系列で追跡可能
- モーターの調子が上向きか下向きかのトレンドを把握

### モーター勝率 vs 選手勝率の比較

- グラフ上でモーターの節成績と使用選手の使用前6ヶ月勝率を並列表示
- **差がプラス**: モーターが選手の実力以上の成績を引き出している（好モーター）
- **差がマイナス**: モーターが選手の足を引っ張っている（悪モーター）
- この比較が貢献ポイント（貢献P）の計算基盤となっている

## 貢献ポイント（貢献P）の詳細

### 定義

モーターの連対率から選手の実力（使用前勝率）を差し引いた、ボートレース日和独自の指標。

### 計算の考え方

- 節別の1着率や連対率は番組構成（インが多いか少ないかなど）に影響される
- 選手の実力差も大きく影響する
- これらの要因を排除し、純粋なモーター性能を数値化したもの

### 解釈

| 貢献Pの値 | 解釈 |
|-----------|------|
| 大きい（プラス方向） | モーターが選手の勝率を上げている。性能の良いモーター |
| 小さい（マイナス方向） | モーターが選手の足を引っ張っている。性能の悪いモーター |
| ゼロ付近 | モーターの影響は中立。選手実力通りの成績 |

### 活用方法

- 2連率が高くても、A1級選手ばかりが使っていたなら当然の結果
- 貢献Pが高ければ、選手の実力以上にモーターが貢献している証拠
- B1級選手が使っても好成績 = 貢献Pが高い = 本物の好モーター

## 超展開データとの連携

超展開データはモーター一覧から直接アクセスする機能ではないが、出走表のモーター情報と組み合わせて使うことで、より精度の高い展開予測が可能になる。

### 連携の流れ

1. **モーター一覧で性能評価**: 貢献P、2連率、勝率でモーターの絶対的な性能を把握
2. **使用履歴で信頼性確認**: 出走数が十分かどうか、強い選手が使って悪い成績ではないか確認
3. **部品交換履歴でトレンド確認**: 最近の部品交換で性能が変化していないか確認
4. **超展開データで1マーク展開予測**: そのモーターを使う選手が1マークでどう攻めるかを予測
5. **決まり手データで勝ちパターン確認**: モーターの特性（出足型/伸び型）と選手の攻めパターンの相性を確認

### MyDataでの統合表示

出走表のMyData設定で以下をまとめて表示可能:
- モーター通算成績
- モーター直近1ヶ月成績
- モーター通算貢献P
- モーター部品交換履歴
- 中間整備情報
- 超展開データ

## 期間別分析

### モーターの集計期間

各競艇場のモーターは年1回抽選で入れ替えが行われる（各場で時期が異なる）。モーターの成績データはこの抽選日からの通算データとなる。

### 分析可能な期間区分

| 期間 | 説明 |
|------|------|
| 通算 | モーター抽選日から現在までの全期間 |
| 直近1ヶ月 | 直近1ヶ月間の成績 |
| 節ごと | 各開催節（4〜7日間）単位の成績 |
| 過去3節比較 | 直近3節の成績を新聞形式で比較表示 |
| 過去10走比較 | 直近10走の結果を比較表示 |

### モーター過去3節比較表

- 2022年追加の機能
- 直近3節の使用選手、勝率、連対率を新聞形式で横並び比較
- MyDataにも登録可能
- モーターの直近トレンドを素早く把握するのに有用

### 出走数の重要性

- データ上優れたモーターでも出走数が少ない場合、信頼度が低い
- モーター抽選直後は出走数が少なく、データの信頼性が低い
- 一般的に出走数30走以上でデータの信頼性が高まるとされる

## BoatAIへの実装示唆

### 短期的に活用可能なデータ

1. **モーター2連率・勝率の取得**: BoatAIの予測モデルにモーター成績を組み込む。既にレース結果データから計算可能な可能性がある
2. **部品交換情報の反映**: 部品交換が多いモーターにペナルティを与える、または交換後の成績変動を追跡するロジック
3. **出走数による信頼度補正**: モーターデータの出走数が少ない場合は重みを下げるロジック

### 中期的に検討すべき機能

1. **貢献ポイント相当の指標実装**: 選手の期待値とモーター実績の差分を計算する独自指標。BoatAIが既に持つ選手勝率データとモーター成績データで計算可能
2. **モーターランキングページの公開**: 各競艇場のモーターランキングをBoatAIサイト上で公開。SEO効果と集客が期待できる
3. **予測へのモーター重み自動調整**: モーター抽選からの経過期間に応じて、予測におけるモーター重みを自動調整するロジック

### BoatAIの差別化ポイント

- ボートレース日和は「モーターデータの閲覧」が中心だが、BoatAIは「モーターデータをAI予測に自動統合」する点で差別化可能
- 貢献ポイントを人間が見て判断する代わりに、AIモデルが自動的にモーター性能を予測に反映
- 部品交換後の成績変動パターンを学習し、交換直後の予測精度を向上させるアプローチ
- モーターの経年劣化パターン（抽選直後 vs 半年後 vs 抽選直前）をモデルに組み込むことで、時期による予測精度の変動を軽減
