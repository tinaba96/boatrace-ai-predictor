# 競合サイト分析: ボートレース日和（kyoteibiyori.com）

> **目的**: 機能を把握し、BoatAIへの実装検討材料とする

## サイト概要

- 運営: 株式会社GAIN
- 旧称「競艇日和」→ 2021年に「ボートレース日和」に改名
- 無料（会員登録も無料）
- データ量は競合サイト中で最大級（MyData項目数: 約400）

---

## 機能一覧

### 1. 出走表

レースごとの詳細情報を9つのタブで切り替え表示。

| タブ | 表示内容 |
|------|---------|
| 基本情報 | 全コース総合データ（平均ST、ST順位、展示順位、勝率、2連対率、3連対率） |
| 枠別情報 | コース別成績、新概念データ（決まり手データ含む） |
| モータ情報 | 通算成績、直近1ヶ月データ、貢献度ポイント |
| 今節成績 | 連対率、平均ST、着順、進入ST |
| 直前情報 | 展示終了後に更新、進入状況、展示タイム |
| オッズ検索 | フォーメーション形式での検索 |
| オッズ一覧 | シンプルなオッズ表示 |
| 出目ランク | 直近1年の該当場の出目データ |
| MyData | ユーザーカスタマイズデータ（後述） |

---

### 2. MyData（カスタマイズ出走表）

約400項目から選択し、自分専用の出走表を構成できる機能。

**選択可能なカテゴリ（判明分）:**

| カテゴリ | 主な項目 |
|---------|---------|
| 基本情報 | 表メモ（場別・モーター・選手）、基本フライング（前期・今期・未消化） |
| モーター | 直近1ヶ月詳細、通算貢献度ポイント、部品交換履歴 |
| 選手パフォーマンス | 枠別平均ST、ST順位、枠別決まり手（総合・数） |
| 連対率・着順 | 枠別1着率・2着率・3着率（女子戦/SG・G1/当地 別に細分化） |
| シミュレーション | 逃げシミュレーション |
| 今節 | 今節得点率早見表、今節成績 |
| 直前 | 展示情報、天候状況 |
| 決まり手 | 新概念データ、超展開データ |

---

### 3. 新概念データ / 超展開データ

公式サイトには掲載されていないオリジナルデータ。

| データ | 内容 |
|--------|------|
| 決まり手データ | 選手が該当コースでどのように勝ったかを数値化（逃げ、差し、捲り、捲り差し等） |
| 超展開データ | 選手の1マークの攻め方を可視化、1着を獲った/逃した方法を分析 |
| 負けた時の決まり手 | 2〜6コースの負けパターン一覧 |

グレード別切り替え可能: 総合 / 一般戦 / SG・G1 / 女子戦

---

### 4. アラート機能

特定条件に合致するレースを自動検出。

| アラート | 条件 | 用途 |
|---------|------|------|
| ガチガチレース検索 | 1号艇逃げ率（50〜75%以上）+ 2号艇逃し率（35〜60%以上） | 堅い決着のレース発見 |
| 穴レース検索 | 1号艇逃げ率（10〜50%以下）+ 2〜6号艇差し・捲り率（20〜40%以上） | 荒れるレース発見 |
| まくりアラート | 展示タイム差（0.07〜0.10以上）で内側艇/全艇と比較 | 捲り候補の検出 |
| 前づけアラート | 展示航走での進入変化を検出 | コース変更の把握 |
| チルト跳アラート | チルト角度変化（+0.5〜+2以上） | 攻めの姿勢検出 |

すべて「本日/明日」のレースに対して検索可能。

---

### 5. データ分析

| 機能 | 検索条件 | 内容 |
|------|---------|------|
| 決まり手検索 | 期間（6ヶ月/1年）、場、艇番、ST順位 | 決まり手別勝率分析 |
| 連対率検索 | 期間、艇番、ST順位 | 本日/明日レースの連対率 |
| レース分析 | 日別/月別、場、各艇の着順条件、万舟フィルター | 過去レース検索 |
| 勝率ランキング | 場、級別（A1/A2/B1/B2）、コース、期間（3ヶ月〜1年） | 勝率/複勝率/3連対率/平均ST/差し/捲り/捲差 |
| 高配当オッズランキング | 日別/月別、勝式（3連単/3連複/2連単/2連複/拡連複/複勝/単勝） | 高配当レース一覧 |
| 優勝履歴 | 年度、レース種類 | SG/G1/G2/G3/一般戦 |
| SG履歴 | 年度、個別SGレース | SGレース別成績 |
| 番組解析 | 年度（2019〜2025）、グレード | 場別番組傾向 |
| 平均値データ | 場、級別 | 各種統計の平均値 |

---

### 6. 出目分析

| 検索条件 | 選択肢 |
|---------|--------|
| 期間 | From〜To（日付指定） |
| 場 | 全24場 |
| 1着コース | 全コース / 1〜6コース個別 |
| 決まり手 | 差し / 捲り / 捲り差し / 全指定 |
| 勝式 | 単勝 / 2連単 / 3連単 |
| 進入固定 | 進入変化しなかったレースのみ |

各号艇（1〜6）+ その他の統計データを表示。

---

### 7. 選手一覧・検索

**検索方法:**
- 名称検索（テキスト）
- 登録番号検索
- 五十音順検索（あ〜わ）
- 期別検索（32期〜137期）
- 支部検索（18支部）

**特殊一覧:**
- ルーキー成績一覧（132期〜、直近3ヶ月）
- 勝率上昇選手一覧（直近1ヶ月 vs 前3ヶ月の勝率比較）
- 勝率下降選手一覧

**選手詳細情報:**
- 基本情報、期別成績、出走予定
- 枠別決まり手、コース別成績
- 直近10走（SG/G1・混合戦の絞り込み可）
- 過去モーター使用選手のコメント履歴

---

### 8. モーター一覧

**表示データ:**
- 出走数、着順（1着〜3着、着外）
- 1着率、2着率、3着率、着外率
- 優出回数、優勝回数
- 使用履歴（選手名、級別、使用日時）
- 決まり手データ（逃げ、差し、捲り、捲差、抜き、恵まれ）

**分析機能:**
- グラフ表示（棒グラフ、円グラフ、折れ線グラフ）
- モーター勝率 vs 使用前選手勝率の比較
- 超展開データ（1マークの攻め方可視化）
- 部品交換履歴
- 中間整備記録
- 日付範囲指定による期間別分析
- ソート: 2連率・3連率・勝率・貢献P順

---

### 9. 場情報

**24場すべてに対して以下を提供:**
- 直近3ヶ月の枠別勝率
- 決まり手別勝率

- 天候・水質・売上状況

## BoatAIとの差分（実装検討用）

### BoatAIに既にある機能
- 出走表の基本情報表示
- 選手・モーター情報
- ルールベースの予測

### BoatAIにない主要機能（実装優先度順）

| 優先度 | 機能 | 実装難易度 | 理由 |
|--------|------|-----------|------|
| 高 | アラート機能（ガチガチ/穴/まくり/前づけ/チルト跳） | 中 | ユーザーの舟券購入を直接支援 |
| 高 | 出目分析 | 中 | 場ごとの傾向把握に必須 |
| 高 | 決まり手データ（新概念データ） | 高 | 差別化の核心機能 |
| 中 | MyData（カスタマイズ出走表） | 高 | ユーザー体験の差別化 |
| 中 | トップページランキング（固い場/荒れている場/イン逃/万舟率） | 低 | 即時実装可能 |
| 中 | 勝率ランキング | 中 | データ分析の基本機能 |
| 低 | 売上履歴・発売票数 | 中 | データソースの確保が課題 |
| 低 | 舟券メモ・回収率分析 | 中 | ユーザー管理機能が必要 |
