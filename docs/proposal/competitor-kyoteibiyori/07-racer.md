# 選手一覧・検索機能の詳細分析

## 概要

ボートレース日和（kyoteibiyori.com）の「選手一覧・検索」機能は、全ボートレーサーのデータベースを提供する機能群である。URL: https://kyoteibiyori.com/racer/

5種類の検索方法を備え、選手の基本情報から詳細な成績データまで一元的にアクセスできる。会員登録不要で基本機能は利用可能だが、メモ機能やMyDataカスタマイズは無料会員登録が必要。

---

## 検索方法の全詳細

### 名称検索

- テキスト入力フィールドに選手名を入力して検索
- 部分一致で検索可能（姓のみ、名のみでも検索できる）
- 検索結果には登録番号、名前、階級（A1/A2/B1/B2）、本日出走予定が表示される
- 女性選手には女性アイコンが付与される

### 登録番号検索

- 選手の登録番号（4桁）を直接入力して検索
- 番号が一致する選手の詳細ページに直接遷移

### 五十音検索

- 「あ」から「わ」までの五十音カテゴリで選手を閲覧
- 各行（あ行、か行、さ行...）ごとにタブが用意されている
- 姓の読みで分類されている
- 選択した行に該当する全選手が一覧表示される

### 期別検索

- 養成所の訓練期（卒業期）で選手を絞り込む
- 対応範囲: 32期〜137期（最新期まで対応）
- 期を選択すると、該当期の全選手が一覧表示される
- 現役選手のみが表示対象

## 特殊一覧

### ルーキー成績一覧

最新5期分（例: 132期〜137期）の若手選手について、直近3ヶ月間の成績を集計して表示する機能。

**表示データ項目:**

| 項目 | 説明 |
|------|------|
| 登録番号 | 選手の登録番号 |
| 選手名 | 名前（女性アイコン付き） |
| 期 | 養成所卒業期 |
| 級別 | A1/A2/B1/B2 |
| 支部 | 所属支部 |
| 出走数 | 集計期間内の出走回数 |
| 1着数 | 1着の回数 |
| 2着数 | 2着の回数 |
| 3着数 | 3着の回数 |
| 勝率 | 着順点合計 / 出走回数 |
| 2連率 | 2着以内率 |
| 3連率 | 3着以内率 |

- 集計期間: 直近3ヶ月（例: 2025年11月15日〜2026年2月15日）
- 集計期間はアクセス日によって自動更新される
- ルーキーの成長度合いを把握するのに有用

### 勝率上昇選手一覧

直近1ヶ月の勝率が前3ヶ月の勝率を上回っている選手をランキング形式で表示する機能。

**抽出条件:**
- 直近1ヶ月の出走数が15走以上
- 直近1ヶ月勝率 > 前3ヶ月勝率

**表示データ項目:**

| 項目 | 説明 |
|------|------|
| 順位 | 上昇幅のランキング |
| 登録番号 | 選手の登録番号 |
| 選手名 | 名前 |
| 級別 | 現在の級別 |
| 支部 | 所属支部 |
| 前3ヶ月勝率 | 直近1ヶ月より前の3ヶ月間の勝率 |
| 直近1ヶ月勝率 | 直近1ヶ月間の勝率 |
| 上昇幅 | 直近1ヶ月勝率 - 前3ヶ月勝率 |

- 対象選手数は変動（例: 158名）
- 調子が上向いている選手の発見に活用
- 舟券購入時の狙い目選手を特定するための指標

### 勝率下降選手一覧

直近1ヶ月の勝率が前3ヶ月の勝率を下回っている選手をランキング形式で表示する機能。

**抽出条件:**
- 直近1ヶ月の出走数が15走以上
- 直近1ヶ月勝率 < 前3ヶ月勝率

**表示データ項目:**
- 勝率上昇選手一覧と同じ構成
- 下降幅（前3ヶ月勝率 - 直近1ヶ月勝率）でソート

- 対象選手数は変動（例: 119名）
- 不調の選手を避けるための判断材料

## 選手詳細ページの全データ項目

選手名をクリックすると遷移する個別選手の詳細ページ。URL形式: `https://kyoteibiyori.com/racer/racer_no/{登録番号}`

| 選手名 | フルネーム | 磯部誠 |
| 期別 | 養成所卒業期 | 105期 |
| 級別 | 現在のランク | A1 |
| 支部 | 所属支部 | 愛知 |

| 血液型 | A/B/O/AB | A型 |

**特殊成績データ（基本情報タブ内）:**

### 期別成績

- 半期ごと（前期: 5〜10月、後期: 11〜4月）の成績を一覧表示
- 各期の級別（A1/A2/B1/B2）の推移が確認可能
- 勝率、連対率、出走数が表示される
- 数年〜十数年分のデータが蓄積されている

### 枠別決まり手

選手が各コース（1〜6コース）で1着を取った際の決まり手を集計。

**決まり手の種類:**

| 決まり手 | 説明 |
|---------|------|
| 逃げ | 1コースからそのまま逃げ切り |
| 差し | インから差して抜く |
| 捲り（まくり） | アウトから一気にまくる |
| 捲差（まくり差し） | まくりと差しの複合技 |
| 抜き | 2周目以降に抜く |
| 恵まれ | 他艇の事故等で繰り上がり |

**表示形式:**
- コース別に決まり手率（%）と決まり手数を表示
- 集計期間: 直近6ヶ月、直近1年
- グレード別切り替え可能:
  - **総合**: 全レース
  - **一般戦**: 一般戦、G2、G3を集計
  - **SG/G1**: SG、G1のみ集計
  - **女子戦**: 女子選手限定レースのみ集計

**2〜6コースの被攻撃データ（2024年5月追加）:**
- 逃し率（1コースが逃した率）
- 差され率
- 捲られ率
- 捲り差され率

### コース別成績

**コース別全艇成績:**
- 該当コースに入った時の自艇と他艇の成績を表示
- 1〜6コースそれぞれの勝率、1着率、2連率、3連率
- グレード別切り替え: 総合 / 一般戦 / SG・G1 / 女子戦

**場別成績:**
- 全24競艇場での成績を一覧表示
- 各場ごとの勝率、複勝率、ST平均などを表示
- 得意場・不得意場の把握に有用

### 直近成績（直近10走）

3種類の直近10走表示に対応。

**1. 枠別直近10走:**
- 該当コースの直近6ヶ月内の直近10走を表示
- 「総合」「当地」の切り替え可能
- 表示項目: 日付、場、レース番号、コース、進入、ST、着順、決まり手

**2. 艇番別直近10走:**
- 該当艇番の直近6ヶ月内の直近10走を表示
- 進入がどのように変わったかを確認可能
- 「総合」「当地」の切り替え可能

**3. 進入別直近10走:**
- 実際の進入コース別の直近10走
- 枠番と進入コースの差異を確認するのに有用

**枠別直近6ヶ月一覧:**
- 直近10走だけでなく、6ヶ月分すべてのレース結果を閲覧可能
- バージョンアップで追加された拡張機能

### 優出一覧

- 直近2年間の優勝戦（優出）の一覧を表示
- 優勝戦に進出した日付、競艇場、着順、使用モーターなどが確認可能
- 優出頻度やパフォーマンスの把握に有用

### コメント履歴

- 節間の選手コメントを一覧で閲覧できる機能（2023年5月追加）
- アクセス経路: 出走表 > 直前情報 > コメント履歴
- 提供している競艇場のみ対応（全場対応ではない）
- 選手の調子やモーターの手応えに関するコメントが時系列で確認可能

### 超展開データ

- 各選手が1マークでどんな攻め方をして、どれだけ勝っているかを数値化
- どんな攻められ方をして、どれだけ負けているかも把握可能
- 1コースの抵抗した回数、2〜6コースの絞った回数などを記録
- 該当コースで1度も勝っていない選手でも攻め方のパターンを予測可能
- 当日レースのみ無料公開（全データはYouTubeメンバーシップで提供）
- アクセス経路: 出走表 > 枠別勝率 > 下部の「超展開データ」ボタン

### 新概念データ

- 枠別情報のタブ内から確認可能
- 各コースの新概念データ履歴として、過去レースにおける他艇の状況を並列表示
- 選手の実力を相対的に評価するための独自指標

### 前づけデータ

- 展示航走で前づけして進入が変化したレースを検索できる機能
- 進入固定時の実績を可視化
- アラート機能として展示航走での前づけ発生時に通知

### メモ機能

- 選手ごとに独自の分析メモを保存可能（無料会員登録が必要）
- 出走表の選手名をタップ > 選手詳細情報 > メモ欄に記入
- 選手やモーターの特徴を記録しておける
- 回収率順での並び替えにも対応

### 出走予定

- 今後の出走スケジュールが一覧表示される
- 競艇場名、開催日程、グレード（SG/G1/G2/G3/一般戦）が表示

### お気に入り・MyData連携

- お気に入り選手として登録可能
- MyData機能で出走表の表示項目をカスタマイズ
- 300個以上の予想材料から表示したい項目を選択可能

## BoatAIへの実装示唆

### 短期的に実装可能な機能

1. **選手検索機能**: 名前・登録番号での基本検索は実装しやすい。Supabaseのracersテーブルを活用
2. **選手詳細ページ**: 基本情報と期別成績は既存データで構築可能
3. **コース別成績表示**: 既に予測に使用しているデータを選手ページとして公開するだけ

### 中期的に検討すべき機能

1. **勝率上昇/下降選手一覧**: 予測精度向上に直結するデータ。日次バッチで勝率推移を計算し、上昇中の選手を優先的に評価するロジックに組み込める
2. **ルーキー成績一覧**: ルーキー戦予測のための特化データ
3. **決まり手データ**: 展開予測の精度向上。コース別・グレード別の決まり手率は1マーク展開予測に不可欠
4. **枠別直近10走**: 直近のフォーム・調子を反映した短期パフォーマンス指標

### BoatAIの差別化ポイント

- ボートレース日和は「データ提供」に特化しているが、BoatAIは「AI予測」を提供する点で差別化可能
- 勝率上昇/下降データを予測モデルに組み込むことで、人間が手動で判断するよりも効率的な予測が可能
- 超展開データのような「展開予測」をAIモデルで自動化することが将来的な方向性
- ボートレース日和が300+のデータ項目をユーザーに選ばせる方針に対し、BoatAIは重要な指標をAIが自動選定して提示するアプローチが有効
