# 場情報機能の詳細分析

## 概要

ボートレース日和の「場情報」は、全国24場の統計データを集約した機能で、`/place/` 配下に配置されている。各場の枠別勝率・決まり手別勝率・今節成績・選手コメント等を、天候やグレード条件で絞り込みながら確認できる。過去2年分のデータを基にしており、公式サイトでは確認しにくい統計を一画面に集約している点が最大の強み。

URL: `https://kyoteibiyori.com/place/`

## 場一覧の表示

### レイアウト

- トップに「場選択」セクションがあり、全24場がリンク形式で一覧表示
- 場名をタップすると各場の詳細ページ（`place_info.php?place_no={場コード}`）に遷移
- 場コードは公式と同一（1:桐生 〜 24:大村）

### 一覧での表示情報

- 場名のリスト表示のみ（一覧画面では統計サマリーは表示しない）
- 各場へのリンクが主要な導線

## 個別場の詳細データ

### 枠別勝率（直近データ）

各場のページでは、1号艇〜6号艇のコース別成績を以下の項目で表示する。

| 項目 | 説明 | 表示例 |
|------|------|--------|
| 進入数 | 該当コースへの進入回数 | 1コース: 2088回 |
| 1着率 | 1着になった割合 | 1号艇: 46.7% |
| 2着率 | 2着になった割合 | 各コース別に表示 |
| 3着率 | 3着になった割合 | 各コース別に表示 |
| ST（スタートタイミング） | 平均スタートタイム | 0.14〜0.17秒 |
| ST順 | 平均スタート順位 | 2.87〜3.94 |

平和島の例では、1号艇1着率46.7%から6号艇1着率3.1%まで段階的に低下するデータが確認できた。

### 決まり手別勝率

選手が該当コースでどのようにして勝ったかを数値化したデータ。1マークの展開予想に活用する。

| 項目 | 説明 | 表示例（平和島1コース） |
|------|------|------------------------|
| 逃げ率 | 逃げ切りの割合 | 43.7% |
| 逃がし率 | 逃げを許した割合 | - |
| 差し率 | 差しの割合 | 16.2% |
| 差され率 | 差されの割合 | - |
| 捲り率 | まくりの割合 | - |
| 捲られ率 | まくられの割合 | - |
| 捲り差し率 | まくり差しの割合 | - |
| 捲られ差し率 | まくり差しされた割合 | - |

**レースグレード別の集計**:
- 総合（すべてのレース）
- 一般戦（一般戦、G2、G3）
- SG/G1（SG、G1）
- 女子戦（女子のみの出走レース）

**期間別データ**:
- 直近6ヶ月
- 直近1年

### 今節成績

- 節別枠別勝率を表示
- 各レースの選手情報を「5121 A1 定松勇樹 1枠」のような形式で記載
- 級別（A1, A2, B1, B2）付きで選手名と枠番を表示

### 選手コメント

- 直前の選手コメントが場情報ページから参照可能
- 具体的には出走表画面からアクセスする導線が主

### 天候・水質情報によるフィルタリング

場情報ページでは、天候や水面状況を条件にした統計絞り込みが可能。

| フィルター項目 | 選択肢 |
|---------------|--------|
| 天候 | 晴れ、曇り、雨 |
| 風向 | 指定可能 |
| 風速 | 指定可能 |
| 波高 | 指定可能 |
| 年度 | 2024年度以降遡及可能 |

これにより「雨の日の平和島で1号艇の1着率がどう変化するか」「強風時のまくり率」といった条件付き統計を確認できる。

### グレード別フィルター

| 選択肢 | 対象レース |
|--------|-----------|
| 一般戦・G3 | 一般戦とG3レース |
| G2・G1・SG | 上位グレードレース |
| すべて | 全グレード |

### その他の表示セクション

- **選手ランキング**: 当該場での成績上位選手
- **モーター情報**: 場ごとのモーター成績
- **優勝者一覧**: 過去の優勝戦結果

### 場情報で確認できる統計サマリー（検索時）

過去2年のデータを基に、以下の指標を場単位で確認可能:

| 指標 | 説明 |
|------|------|
| 平均払い戻し | 3連単の平均配当金額 |
| 1号艇の1着率 | イン勝率 |
| 逃げ率 | 1号艇の逃げ成功率 |
| 万舟率 | 10,000円以上配当の出現率 |
| 決まり手の勝数 | 決まり手別の勝利数 |
| コース毎の1〜3着数 | 各コースの着順分布 |

## BoatAIへの実装示唆

### 優先度: 高

1. **場別統計ダッシュボード**: 各場の枠別勝率・決まり手別勝率を表示するページを新設。BoatAIの既存分析データ（`data/analysis/venue-{コード}/`）と連携し、ルール根拠の可視化にも活用できる

2. **天候条件付き統計**: Supabaseに天候データを蓄積済みであれば、天候別のコース勝率をフィルタリング表示できるようにする。予想精度の向上にも直結する

3. **グレード別フィルター**: 一般戦とSG/G1では傾向が大きく異なるため、グレード別の統計切り替えは予想ロジックにも重要

### 優先度: 中

4. **今節成績表示**: 現在の開催における選手成績をリアルタイムに表示。予想ページに統合可能

5. **場別の特徴サマリー**: 各場の「水面特性」（潮の影響、風の傾向など）をテキストで表示する情報ページ

### 差別化ポイント

- ボートレース日和は「生データの網羅性」が強み。BoatAIは「AIによる分析・予想」が強みなので、場情報は予想の根拠説明として活用するのが効果的
- 例: 「この予想の根拠: 平和島は追い風3m以上で1号艇逃げ率が58%に上昇」のような形で場データを予想に紐づける
