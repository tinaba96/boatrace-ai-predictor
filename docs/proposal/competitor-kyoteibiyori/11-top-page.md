# トップページの詳細分析

## 概要

ボートレース日和のトップページは、当日の全場開催状況とリアルタイムランキングを一画面に集約した「ダッシュボード型」の設計。ユーザーがアクセスした瞬間に「今日どの場が固いか/荒れているか」を直感的に把握でき、そこから出走表や場情報への導線を提供している。レース検索ツール（ガチガチレース検索・穴レース検索・各種アラート）も備え、目的別の高速アクセスを実現している。

URL: `https://kyoteibiyori.com/`

## リアルタイムランキング

トップページ右側（モバイルではスクロール下部）に、当日のレース結果を集計したリアルタイムランキングを4種類表示。

### 固い場ランキング

**概要**: 当日の3連単平均配当が低い（=堅い決着が多い）場のTOP5を表示。

| 表示項目 | 説明 | 表示例 |
|---------|------|--------|
| 順位 | 1〜5位 | 1位 |
| 場名 | ボートレース場名 | 宮島 |
| 平均配当 | 3連単平均払戻金額 | 3,388円 |
| 開催ステータス | 開催日目/終了 | 3日目 |
| 詳細リンク | 場情報への遷移 | あり |

**活用**: 固い場は本命党向け。1号艇の逃げが決まりやすい場の傾向を把握できる。

### 荒れているランキング

**概要**: 当日の3連単平均配当が高い（=波乱が多い）場のTOP5を表示。

| 表示項目 | 説明 | 表示例 |
|---------|------|--------|
| 順位 | 1〜5位 | 1位 |
| 場名 | ボートレース場名 | 児島 |
| 平均配当 | 3連単平均払戻金額 | 11,232円 |
| 最高配当 | 当日最高配当 | 表示あり |
| 開催進捗 | 消化レース数 | あり |

**活用**: 穴狙い派向け。天候や水面変化の影響で荒れている場を素早く発見できる。

### イン逃げ率ランキング

**概要**: 当日の1号艇（1コース）逃げ成功率が高い場のTOP5を表示。

| 表示項目 | 説明 | 表示例 |
|---------|------|--------|
| 順位 | 1〜5位 | 1位 |
| 場名 | ボートレース場名 | 徳山 |
| 逃げ率 | 1コース逃げ成功率 | 91.7% |
| 場名(5位) | 下位場名 | 芦屋 |
| 逃げ率(5位) | 下位逃げ率 | 66.7% |

**活用**: イン逃げ率が高い場は1号艇の単勝・複勝が狙いやすい。BoatAIのルールマッチとの相性が良い。

### 万舟率ランキング

**概要**: 当日の10,000円以上配当（万舟）の出現率が高い場のTOP5を表示。

| 表示項目 | 説明 |
|---------|------|
| 順位 | 1〜5位 |
| 場名 | ボートレース場名 |
| 万舟率 | 万舟出現率（%） |

**活用**: 万舟狙いの場選びに直結。3連単の高配当を狙うユーザー向け。

## レース検索ツール

トップページには、条件を指定してレースを絞り込む5つの検索ツールが配置されている。

### ガチガチレース検索

| 項目 | 説明 |
|------|------|
| 目的 | 堅い決着が想定されるレースを検索 |
| 条件1 | 1号艇の逃げ率（閾値指定） |
| 条件2 | 2号艇の逃がし率（閾値指定） |
| 対象 | 本日/明日のレース |
| 結果 | 条件を満たすレースの一覧 |

### 穴レース検索

| 項目 | 説明 |
|------|------|
| 目的 | 荒れそうなレースを検索 |
| 条件1 | 1号艇の低逃げ率（閾値指定） |
| 条件2 | 2〜6号艇の高差し率・高まくり率 |
| 対象 | 本日/明日のレース |
| 結果 | 波乱の可能性が高いレースの一覧 |

### まくりアラート

| 項目 | 説明 |
|------|------|
| 目的 | まくりが起こりやすいレースを検知 |
| 条件 | 展示タイムの変化等から判定 |
| 表示 | アラート形式で該当レースを表示 |

### 前づけアラート

| 項目 | 説明 |
|------|------|
| 目的 | 前づけ（コース取り）が予想されるレースを検知 |
| 条件 | 選手の進入傾向データから判定 |
| 表示 | アラート形式で該当レースを表示 |

### チルト跳アラート

| 項目 | 説明 |
|------|------|
| 目的 | チルト角度を跳ねている（通常より大きく上げている）選手がいるレースを検知 |
| 条件 | チルト角度データから判定 |
| 表示 | アラート形式で該当レースを表示 |

| 開催日目 | 何日目の開催か | 3日目 |

| 時間帯アイコン | ナイター/モーニング等 | アイコン表示 |

## BoatAIへの実装示唆

### 優先度: 高

1. **場状況リアルタイム表示**: 開催中の全場の進行状況（何R目か、次の発走時刻）をトップページに表示。BoatAIの予想対象レースへの導線として最重要

2. **固い場/荒れている場ランキング**: 当日の平均配当に基づくランキング表示。BoatAIのAI予想と組み合わせて「今日の狙い場」として提示すると付加価値が高い

3. **本日/明日の切り替え**: 翌日のレース情報を前日夜から確認できる機能。予想配信のタイミングとも連動させやすい

### 優先度: 中

4. **レース検索ツール**: ガチガチレース検索や穴レース検索は、BoatAIのルールマッチ機能と統合できる。例えば「BoatAIルールに合致するレース」という検索軸を追加すれば独自の価値になる

5. **イン逃げ率/万舟率ランキング**: BoatAIの分析済み会場データと組み合わせて、「AI推奨の注目場」としてランキング化

### 優先度: 低

6. **各種アラート機能**: まくりアラート・前づけアラート・チルト跳アラートはデータ取得の仕組みが必要。長期的には実装検討の価値あり

### 差別化ポイント

- ボートレース日和のトップページは「データの網羅性」と「リアルタイム性」が強み
- BoatAIは「AI予想の的中実績」「ルールマッチの根拠表示」を前面に出すことで差別化
- トップページに「本日のAI推奨レース TOP5」のような独自コンテンツを配置し、データサイトではなく「予想サービス」としてのアイデンティティを明確にする
