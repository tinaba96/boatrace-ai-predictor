# 🎯 BoatAI - ユーザーバリュー・プロポジション

**BoatAI** は「ボートレースの買い目をAIが予想してくれるサイト」ではなく、「ボートレースの勝ちパターンをAIが統計的に教えてくれるサイト」です。

---

## 📊 現在ユーザーに提供している価値

### 1. **買い目の絞り込み** 🎯

**従来の悩み**:
- ボートレースは6選手中から1-2-3着を予想する必要があり、組み合わせは20,160通り
- 初心者は情報が多すぎて選べない
- ベテランも「この情報をどう組み合わせるか」で試行錯誤

**BoatAI の解決**:
- **3つの予想モデルで候補を3-4艇に絞る**
  - 🏆 **スタンダード**: 全ユーザー向け（最高精度）
  - 💰 **本命狙い**: リスク低く回収重視
  - 🎲 **穴狙い**: 高配当を狙う積極的購入者向け

- **複勝でも単勝でも買える**
  - 複勝は「1着か2着」の広い買い目で的中率高
  - 初心者は複勝から始められる

### 2. **荒れるレースを事前警告** ⚠️

**従来の悩み**:
- 「本命が飛ぶ」という番狂わせはボートレースでよくある
- 新聞やSNS の予想を信じたら穴が出て損失
- 事前に「危ないレース」か「堅いレース」かわからない

**BoatAI の解決**:
- **「インが崩れる確率」を0-100で数値化** 
  - 🟢 スコア 20以下: 本命が走りやすい（堅め）
  - 🟡 スコア 50: 中程度の荒れ
  - 🔴 スコア 80以上: 穴が出やすい（危険）

- **活用例**:
  - スコア 80以上 → 穴狙いモデルで買う / 本命買いは避ける
  - スコア 20以下 → 単勝で本命狙い可能
  - スコア 50 → リスク・リターンのバランス型買い目で

### 3. **当地実績・モーター性能・展示タイムを統合分析** 📈

**従来の手順**:
1. 新聞で選手情報確認 → モーター確認 → オッズ確認 → 手書き予想
2. 本命候補・穴候補ごとに手計算で期待値判定
3. 時間がかかりすぎて「発走直前で勘で買う」になる

**BoatAI の解決**:
- **各艇の「スコア」をワンクリックで確認**
  - 選手成績（全国勝率+当地勝率）
  - モーター性能（2連率）
  - ボート性能（2連率）
  - 展示タイム・展示ST
  - **ターン予測**: 「逃げが利く」「捲りが強い」という展開適性

- **すべて自動集計済みで瞬時に判定可能**

### 4. **個人成績を自動記録** 📊

**従来の悩み**:
- 買ったレースの的中・外れを手書きで記録
- 「先月の成績」「会場別成績」を計算するのが手作業
- 回収率100%を超えているルールを見つけたくても、全レースを手動集計するのは不可能

**BoatAI の解決**:
- **サイト内での購入検討レースが自動的に記録される**（将来機能）
- **的中率・回収率を自動集計**
  - 単勝・複勝・3連複・3連単ごと
  - 日別・月別・会場別の統計
  - 「どのモデル（本命狙い/穴狙い）が自分に合っているか」を可視化

2. **オッズ × 予想の組み合わせ**
   - 買い目の期待値を計算
   - 「買う / 見送る」の線引き

## 📞 よくある質問

### Q1: 「100% 当たるの？」

**A**: いいえ。ボートレースはそもそも100% 予想が当たらないスポーツです。
- 展示が良い選手が本番で失敗することもある
- 急なハンドル故障もあり得る

BoatAI の役割は「的中率を最大化する」「損を最小化する」ことです。

### Q2: 「新聞の予想とどっちが良い？」

**A**: 別物として使い分けるのが最適です。

| 視点 | 新聞 | BoatAI |
|------|------|--------|
| 速報性 | △（朝配信） | ✅（5分ごと更新） |
| 信頼度 | ✅（有名記者） | ✅（統計ベース） |
| 複数視点 | ✅（複数記者） | ✅（3モデル） |
| 定量化 | △（言葉のみ） | ✅（スコア化） |

→ **両方をチェックして判定する** のが上級者の買い方

### Q3: 「本当に儲かる？」

**A**: ボートレースは「必ず儲かる」ものではありません。ただ：

- スタンダードモデルで複勝買い: 的中率 50% → **期待値は+10%**
- 本命狙いモデル: **より堅く、ローリスク**
- 穴狙いモデル: **高配当が出る可能性**

⚠️ **大事なこと**:
- 1レースで儲けようとしない（複数レース購入時に効果）
- 「今日は買わない」の判断も重要（スコア50以上は見送り）
- 資金管理が最重要（1レース に全額を賭けない）

### Q4: 「初心者でも使える？」

**A**: はい。以下の順序で始めてください：

1. **ルール学習**: YouTube で「ボートレース基本ルール」（5分）
2. **複勝練習**: スコア 20 以下のレースで複勝を買う
3. **AI理解**: BoatAI の「本命」をクリックして根拠を読む
4. **パターン学習**: 月3-5レース購入して「自分の得意パターン」発見
