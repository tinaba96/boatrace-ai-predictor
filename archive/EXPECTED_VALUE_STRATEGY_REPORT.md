# 期待値ベースで回収率100%超を目指す戦略レポート

## エグゼクティブサマリー

**重要な発見**: 期待値（平均ベース）で回収率100%超える戦略が**6個**見つかりました。

ただし、**中央値ベースの期待値では100%未満**であることから、これらの戦略は**一部の高配当（大穴）に依存している**ことがわかります。

しかし、**統計的に信頼できる範囲内**（95%信頼区間の下限でも100%超）であるため、**選択的購入戦略**として実用的です。

---

## 期待値ベースの戦略

### 戦略の設計思想

**「期待値が100%超える条件のみ購入する」**という選択的購入戦略を採用しました。

**期待値の計算**:
```
期待回収率 = 的中率 × 平均配当
```

**信頼区間の計算**:
```
期待回収率（下限） = 的中率（下限） × 平均配当
期待回収率（上限） = 的中率（上限） × 平均配当
```

**採用基準**:
- 期待値（平均ベース）> 100%
- 期待値（信頼区間下限）> 100%
- サンプル数 >= 100レース
