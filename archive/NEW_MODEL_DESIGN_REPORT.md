# 回収率100%超を目指す新モデル設計レポート

## エグゼクティブサマリー

**重要な発見**: 大穴依存を排除した場合、**中央値ベースの期待回収率で100%超を達成する戦略は見つかりませんでした**。

これは、ボートレースの購入において、**大穴的中に依存せずに回収率100%超を達成するのは、統計的に極めて困難**であることを示しています。

---

## 分析アプローチ

### 1. 既存モデルの分析

既存の3つのモデル（standard, safeBet, upsetFocus）について、外れ値を除外した回収率を分析しました。

#### standardモデル
- **複勝**: 回収率89.98% → 外れ値除外後66.88%
- **3連単**: 回収率101.87% → 外れ値除外後46.32%（大穴依存）

#### safeBetモデル
- **複勝**: 回収率91.36% → 外れ値除外後65.85%
- **3連単**: 回収率68.27% → 外れ値除外後39.78%

#### upsetFocusモデル
- **複勝**: 回収率88.32% → 外れ値除外後63.29%
- **3連単**: 回収率75.67% → 外れ値除外後29.03%（大穴依存）

**発見**: すべてのモデルで、外れ値を除外すると回収率が100%を下回ります。

### 2. 中央値ベースの期待回収率分析

大穴依存を完全に排除するため、**中央値ベースの期待回収率**で評価しました。

**計算式**:
```
期待回収率 = 的中率 × 中央値配当
```

この方法では、異常に高い配当（大穴）の影響を完全に排除できます。

1. **全条件**

3. **1号艇がA2級以上**
4. **1号艇のモーター2連率40%以上**

6. **A1級2名以上**

8. **複合条件**（例: 1号艇がA1級 かつ モーター2連率40%以上）
9. **会場別条件**（24会場 × 複合条件）

**結果**: **回収率100%超の戦略は0個**

## 数学的根拠

### なぜ回収率100%超が困難なのか

1. **配当の構造**
   - ボートレースの配当は、人気度に応じて設定される
   - 人気が高い（的中しやすい）ほど配当は低くなる
   - 人気が低い（的中しにくい）ほど配当は高くなる

2. **期待値の計算**
   ```
   期待回収率 = 的中率 × 平均配当
   ```
   - 的中率が高い → 配当が低い → 期待回収率は低い
   - 的中率が低い → 配当が高い → 期待回収率は低い（的中しにくいため）
   - **最適なバランス点でも、期待回収率は100%未満になる傾向がある**

3. **大穴依存の問題**
   - 回収率100%超を達成するには、異常に高い配当（大穴）に依存する必要がある
   - しかし、大穴は**再現性が低く、統計的に信頼できない**

## 数学的根拠の詳細

### 期待回収率の計算

#### 平均ベース
```
期待回収率 = 的中率 × 平均配当
```

#### 中央値ベース（大穴依存を排除）
```
期待回収率 = 的中率 × 中央値配当
```

### 統計的有意性の検証

#### 信頼区間の計算
```
95%信頼区間 = 的中率 ± 1.96 × √(的中率 × (1 - 的中率) / サンプル数)
```

#### 期待回収率の信頼区間
```
期待回収率（下限） = 的中率（下限） × 中央値配当
期待回収率（上限） = 的中率（上限） × 中央値配当
```

### 外れ値の除外方法

#### IQR法（四分位範囲法）
```
IQR = Q75 - Q25
下限 = Q25 - 1.5 × IQR
上限 = Q75 + 1.5 × IQR
```

この方法で、異常に高い配当（大穴）を外れ値として除外しました。
