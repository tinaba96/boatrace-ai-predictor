# ボートレース最適購入戦略導出プロンプト

## プロンプト本文

あなたは、ボートレースの過去データから数学的に最適な購入戦略を導出する統計分析の専門家です。以下のデータと要件に基づいて、**どのような統計情報の時に、どのように買えば良いのか**を数学的に明確に示してください。

---

## データ構造

### レースデータ（races.json）
各レースには以下の情報が含まれています：
- **レース情報**: 日付、会場コード、レース番号、天候、風向・風速、波高、気温、水温
- **選手情報（6艇分）**:
  - `lane`: 枠番（1-6）
  - `name`: 選手名
  - `grade`: 級別（A1, A2, B1, B2）
  - `age`: 年齢
  - `globalWinRate`: 全国勝率（%）
  - `global2Rate`: 全国2連率（%）
  - `localWinRate`: 当地勝率（%）
  - `local2Rate`: 当地2連率（%）
  - `motorNumber`: モーター番号
  - `motor2Rate`: モーター2連率（%）
  - `boatNumber`: ボート番号
  - `boat2Rate`: ボート2連率（%）

### 予想・結果データ（predictions/*.json）
各レースには以下の情報が含まれています：
- **予想情報**: 
  - `prediction.topPick`: 本命予想（枠番）
  - `prediction.top3`: 3連複予想（[枠番1, 枠番2, 枠番3]）
  - `prediction.players`: 各選手のAIスコアと詳細情報
- **結果情報**:
  - `result.rank1`, `rank2`, `rank3`: 実際の着順
  - `result.payouts`: 配当情報
    - `win[枠番]`: 単勝配当
    - `place[枠番]`: 複勝配当
    - `trifecta[組み合わせ]`: 3連複配当
    - `trio[組み合わせ]`: 3連単配当

## 分析タスク

### 1. 統計的パターンの発見

以下の観点から、**統計的に有意なパターン**を発見してください：

#### A. 単一因子分析
- **枠番別の勝率・複勝率・3着内率**とその統計的有意性（p値、信頼区間）
- **級別（A1/A2/B1/B2）の勝率**とその統計的有意性
- **年齢層別の勝率**（20代前半/後半、30代前半/後半、40代、50代以上）
- **全国勝率の階層別勝率**（3%未満、3-4%、4-5%、5-6%、6-7%、7%以上）
- **モーター2連率の階層別勝率**（0-30%、30-35%、35-40%、40-45%、45%以上）
- **ボート2連率の階層別勝率**
- **当地アドバンテージ**（localWinRate - globalWinRate）の効果
- **天候・風向・波高**などの環境要因の影響

#### B. 複合因子分析
以下の組み合わせで、**相乗効果**があるかを検証：
- 級別 × 枠番（例：A1級 × 1号艇）
- 級別 × モーター2連率（例：A1級 × 好モーター40%以上）
- 枠番 × モーター2連率（例：1号艇 × 好モーター）
- 当地アドバンテージ × 級別
- 外枠（4-6号艇） × 超好モーター（45%以上）
- その他の有意な組み合わせ

#### C. 条件別の回収率分析
以下の購入方法について、**条件別の回収率**を計算：
- **単勝購入**: 本命予想に100円投資した場合の回収率
- **複勝購入**: 本命予想に100円投資した場合の回収率
- **3連複購入**: 予想top3に100円投資した場合の回収率
- **3連単購入**: 予想top3を順番通りに100円投資した場合の回収率

各購入方法について、以下の条件で分類して回収率を計算：
- 会場別
- 級別構成（A1級の有無、B2級の有無など）
- モーター2連率の分布（好モーターの有無、好モーターの数など）
- 当地アドバンテージの有無
- 天候条件
- その他の有意な条件

### 2. 最適購入戦略の導出

#### A. 条件別最適購入ルール
以下の形式で、**数学的に最適な購入ルール**を提示してください：

```
【条件パターン1】
統計情報の条件:
- 会場: [会場名]
- 級別構成: [例: A1級1名以上、B2級2名以下]
- モーター2連率: [例: 40%以上のモーターが2つ以上]
- 当地アドバンテージ: [例: 2%以上の選手が1名以上]
- その他: [天候、風向など]

最適な購入方法:
- 購入種別: [単勝/複勝/3連複/3連単]
- 購入対象: [本命のみ/本命+2着候補/3連複/3連単]
- 期待回収率: [X%]
- 統計的信頼度: [95%信頼区間: X% - Y%]
- サンプル数: [Nレース]

数学的根拠:
- この条件での的中率: [X%]
- 的中時の平均配当: [Y倍]
- 期待値: [(X% × Y倍 - 100%)]
- 統計的有意性: [p値 < 0.05 または 信頼区間が100%を上回る]
```

#### B. リスク調整済み最適戦略
各条件パターンについて、以下を計算：
- **シャープレシオ**: （期待回収率 - 100%）/ 回収率の標準偏差
- **最大ドローダウン**: 連続して負けた場合の最大損失率
- **ケリー基準**: 最適な投資比率（f = (p × b - q) / b）
  - p: 的中確率
  - b: 的中時の配当倍率
  - q: 外れる確率（1 - p）

#### C. 条件別購入判断フローチャート
以下のような判断フローチャートを作成：
```
1. 会場を確認
   → 会場別の最適モデル・購入種別を適用

2. 級別構成を確認
   → A1級が2名以上 → 本命狙い戦略
   → B2級が3名以上 → 穴狙い戦略
   → その他 → スタンダード戦略

3. モーター2連率を確認
   → 40%以上のモーターが2つ以上 → 好モーター重視
   → 45%以上のモーターが1つ以上 → 超好モーター単独狙い

4. 当地アドバンテージを確認
   → 当地勝率が全国勝率+2%以上の選手 → 当地重視

5. 最終判断
   → 上記条件を組み合わせて最適な購入方法を決定
```

### 3. 数学的検証

#### A. 統計的有意性の検証
各パターンについて、以下を検証：
- **カイ二乗検定**: 期待値と実際の結果の差が統計的に有意か
- **t検定**: 回収率が100%を上回るか（片側検定）
- **信頼区間**: 95%信頼区間で回収率が100%を上回るか
- **効果量（Cohen's d）**: 効果の大きさ

#### B. 過学習の検証
- **交差検証**: データを訓練用と検証用に分割して検証
- **時系列検証**: 古いデータで学習し、新しいデータで検証
- **ブートストラップ法**: リサンプリングによる信頼区間の計算

#### C. 再現性の検証
- **日別の回収率の変動**: 標準偏差、変動係数
- **会場別の再現性**: 会場ごとの回収率の安定性
- **条件別の再現性**: 同じ条件での回収率のばらつき

## 重要な要件

1. **数学的厳密性**: すべての結論は統計的に検証されたものであること
2. **実践的可能性**: 実際の購入時に使える具体的なルールであること
3. **再現性**: 同じ条件で同じ結果が得られること
4. **リスク管理**: 損失を最小化する戦略も提示すること
5. **条件の明確性**: 各ルールの適用条件を数値で明確に示すこと
