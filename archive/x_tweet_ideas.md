## パターン1: モーター2連対率の重要性（専門知識型）

```
【ボートレースで勝つために知っておくべきこと】

「A1級の選手が1号艇なのに負けた...なぜ？」

その答えは**モーター**にあります。

ボートレースでは、選手は抽選でモーターを割り当てられます。
このモーター性能の差が、選手の実力以上にレース結果を左右するのです。

【モーター2連対率とは】
そのモーターを使った過去のレースで、1-2着に入った割合。

【評価基準】
✅ 45%以上: 超優良モーター（即買い候補）
✅ 40-45%: 優良モーター（積極的に狙える）
⚠️ 35-40%: 普通（様子見）
❌ 35%未満: 不調モーター（警戒が必要）

【重要な事実】
実力あるA1級選手でも、モーターが悪いと負けます。
逆に、B1級選手でも優良モーターなら穴になります。

選手の実力 × モーター性能 = 総合力

【実例】
A1級選手（勝率7.5） × 優良モーター（45%）
→ 勝率70%（最強）

A1級選手（勝率7.5） × 不調モーター（28%）
→ 勝率40%（中程度）

B1級選手（勝率5.5） × 優良モーター（45%）
→ 勝率30%（穴候補）

【買い方のコツ】
出走表でモーター2連対率を必ずチェック。
40%以上なら積極的に狙いましょう。

モーター性能を制する者が、ボートレースを制します。

https://boat-ai.jp/

#ボートレース #モーター #2連対率 #ボートレース予想
```

## パターン3: 全国勝率 vs 当地勝率の落とし穴（データ分析型）

```
【ボートレースのデータで最も重要なこと】

「全国勝率が高い選手を買えばいいんでしょ？」

実は、それだけでは不十分です。

【重要な事実】
**当地勝率 > 全国勝率**

場によって水面特性が異なるため、
そのボートレース場での過去成績の方が重要です。

【実例】

選手A:
全国勝率 7.20（優秀）
当地勝率 5.80（戸田が苦手）
→ 戸田では過信禁物

選手B:
全国勝率 6.10（普通）
当地勝率 7.40（びわこが得意）
→ びわこでは要注目

【数値の目安】

全国勝率:
7.00以上: トップクラス
6.00-7.00: 優秀
5.00-6.00: 普通
5.00未満: やや苦戦中

当地勝率:
7.50以上: その場の得意選手
6.50-7.50: 標準的
6.00未満: その場が苦手

【AIの評価方法】

当地勝率を全国勝率より**1.5倍重視**します。
場の相性は実力以上に重要だからです。

【2連対率も重要】

2連対率（1-2着に入る確率）は安定性の指標。

40%以上: 安定して上位（A1級）
30-40%: まずまず（A2-B1級）
30%未満: 苦戦中（B1-B2級）

勝率が高くても2連対率が低いと、
「たまに勝つが凡走も多い」と判定されます。

【総合的な見方】

✅ 級別（A1/A2/B1/B2）
✅ 全国勝率（ベースの実力値）
✅ **当地勝率（最重要）**
✅ **2連対率（安定性）**
✅ **モーター2連対率（最重要）**

これらを総合的に見て予想しましょう。

https://boat-ai.jp/

#ボートレース #データ分析 #当地勝率 #2連対率

## パターン5: 展示航走の見方と最終確認（実践的知識型）

```
【ボートレースで展示航走を見るべき理由】

出走表のデータだけでは不十分です。
**展示航走**で最終確認しましょう。

【展示航走とは】

レース前に全艇が1周する練習走行。
ここでモーター性能を最終確認できます。

【見るべきポイント】

✅ 展示タイム
6.75秒以下: 優秀
6.75-6.85秒: 普通
6.85秒以上: やや遅い

✅ まわり足（ターン性能）
スムーズなターン → ○
外に膨らむ → ×

✅ 伸び足（直線加速）
直線でグングン加速 → ○
伸びが鈍い → ×

【実例】

優良モーター（2連対率42%）:
展示タイム: 6.75秒
→ 直線で他艇を引き離す

不調モーター（2連対率28%）:
展示タイム: 6.92秒
→ 直線で置いていかれる

**0.1秒の差**が、着順に大きく影響します。

【モーター2連対率との相関】

モーター2連対率が高くても、
展示タイムが遅い場合は要注意。

逆に、モーター2連対率が低くても、
展示タイムが速い場合は穴候補。

【選手の調子も見える】

展示航走で選手の調子も分かります。

✅ スムーズなスタート
✅ 安定したターン
✅ 力強い伸び足

→ 調子が良い

❌ スタートが遅い
❌ ターンで外に膨らむ
❌ 伸び足が鈍い

→ 調子が悪い

【最終確認のタイミング】

✅ 出走表でモーター2連対率をチェック
✅ 展示航走で展示タイムを確認
