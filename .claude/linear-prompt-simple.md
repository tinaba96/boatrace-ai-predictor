# Claude Code用 - Linear統合プロンプト（簡潔版）

Claude Codeで実装する際に、必ずLinearチケットを作成・更新してください。

## 📋 実装時の必須手順

### 1. 実装開始時
```bash
# Linearチケットを作成
npm run linear:create "[機能名]" "[詳細説明]"

# 作成されたチケットIDを取得（例: BIZ-123）
# チケットを「進行中」に更新
npm run linear:update BIZ-123 "進行中" "実装を開始"
```

### 2. 実装中
```bash
# 重要な進捗があればコメントを追加
npm run linear:comment BIZ-123 "進捗: [内容]"
```

### 3. 実装完了時
```bash
# チケットを「完了」に更新
npm run linear:update BIZ-123 "完了" "実装完了"

# 実装内容の概要をコメントに記録
npm run linear:comment BIZ-123 "実装完了: [実装内容の概要]"
```

## 🎯 プロンプト例

### 機能実装
```
以下の機能を実装してください。

【機能名】
[機能の説明]

【Linear統合】
実装開始時にLinearチケットを作成し、「進行中」に更新してください。
実装完了時に「完了」に更新し、実装内容の概要をコメントに記録してください。
```

### バグ修正
```
以下のバグを修正してください。

【バグの説明】
[バグの詳細]

【Linear統合】
1. Linearチケットを作成（タイトル: バグ修正: [バグ名]）
2. チケットを「進行中」に更新
3. 原因を調査し、コメントに記録
4. 修正を実施
5. テストを実施
6. チケットを「完了」に更新
```

---

**重要: 実装を開始する前に、必ずLinearチケットを作成してください。**
