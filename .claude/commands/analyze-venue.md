# 会場別予測分析

会場コード **$ARGUMENTS** の分析を実行してください。

## 実行手順

1. **基本統計の確認**
   - レース数、1号艇勝率、荒れ度
   - 単勝/複勝/3連複/3連単の回収率

2. **条件探索**
   - 回収率100%を超える条件を探索
   - 例：「1号艇A1級 + モーター2連率40%以上」

3. **結果保存**
   - `data/analysis/venue-{コード}/` に結果を保存

## スクリプト

```bash
node scripts/analysis/collect-venue-stats.js --venue=$ARGUMENTS
```
