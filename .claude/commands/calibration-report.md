# 展開予測キャリブレーション分析

決まり手予測の確率精度を厳密に評価し、データ蓄積による改善トレンドを確認します。

## 引数

- `$ARGUMENTS`: オプション（例: `--weeks=4`, `--from=2026-03-01 --to=2026-04-01`, `--model=standard`）

## 実行

```bash
node scripts/analysis/calibration-report.js $ARGUMENTS
```

## レポートの見方

### ブライアスコア
- 低いほど確率予測が正確（完璧=0、ランダム≈0.83）
- 週次推移で改善傾向にあるかを確認

### 確率帯別キャリブレーション
- 「予測確率X%」と出したとき、実際にX%の割合で的中しているかを確認
- 乖離が5pt未満なら信頼できる、10pt以上なら補正が必要

### 買い方提案への影響
- 信頼できる確率帯の確率はユーザーの賭け方判断にそのまま使える
- 要改善の確率帯は、補正をかけてから表示すべき
