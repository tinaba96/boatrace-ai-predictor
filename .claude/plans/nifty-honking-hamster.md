# 「正解」→「的中」用語統一プラン

## Context

AdSense審査対策として「的中」「的中率」を「正解」「正解率」に変更していたが、審査合格済みのため元の用語に戻す。CSSクラス名・DB・APIフィールド名は変更なし（UIテキストのみ）。

## 変更対象ファイルと置換ルール

### 置換パターン

| 変更前 | 変更後 |
|--------|--------|
| 正解率 | 的中率 |
| 正解レース | 的中レース |
| 正解数 | 的中数 |
| 正解実績 | 的中実績 |
| 単勝正解 | 単勝的中 |
| 複勝正解 | 複勝的中 |
| 3連複正解 | 3連複的中 |
| 3連単正解 | 3連単的中 |
| 単勝不正解 | 単勝不的中 |
| 複勝不正解 | 複勝不的中 |
| 3連複不正解 | 3連複不的中 |
| 3連単不正解 | 3連単不的中 |
| 正解したレース | 的中したレース |
| 正解しますか | 的中しますか |
| 正解はありません | 的中はありません |
| 正解するか | 的中するか |
| 正解だったか | 的中だったか |
| '正解' (ハッシュタグ) | '的中' |
| 正解 (Headerナビラベル) | 的中 |
