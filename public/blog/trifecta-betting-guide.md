「3連単は当たらない」と思っていませんか？

確かに3連単は120通りもあり、闇雲に買えばまず当たりません。しかし、**軸選び・点数の絞り方・資金配分**を正しく理解すれば、3連単は最も効率よく稼げる舟券です。

## 3連単とは？ — 配当の仕組みを理解する

### 基本ルール

**3連単（3連勝単式）**は、1着・2着・3着を**着順通り**に当てる舟券です。

| 舟券種別 | 組み合わせ数 | 平均配当 |
|---------|------------|---------|
| 3連複 | 20通り | 約2,000円 |
| 3連単 | **120通り** | **約7,000円** |
| 2連単 | 30通り | 約2,000円 |

3連複と比べて組み合わせが6倍ある分、**配当も高くなりやすい**のが最大のメリットです。

### なぜ3連単が有利なのか？

- 的中時のリターンが大きく、**少ない的中回数でもプラス収支**にできる
- 着順まで予想することで、**自分の分析力がダイレクトに反映**される
- 人気の偏りが大きく、**穴目の期待値が高い**レースが存在する

## 買い目の絞り方 — 軸選びが9割

### ステップ1: 1着軸を決める

3連単で最も重要なのは**1着の軸選び**です。1着を固定するだけで、買い目は120通りから20通りに減ります。

**1着軸の選び方：**
- A1級選手がインコースにいるレース
- モーター2連率40%以上の選手
- スタート展示で先行している艇
- BoatAIの予測で1着信頼度が高い選手

### ステップ2: 2着候補を3〜4艇に絞る

1着軸を決めたら、2着候補を絞ります。全5艇から**3〜4艇**に絞るのが理想です。

### ステップ3: 3着候補を決める

2着候補に選ばなかった艇も含めて、3着候補を選びます。

## フォーメーション vs ボックス — どっちが有利？

### フォーメーション買い ✅

**1着・2着・3着をそれぞれ指定する買い方。点数を絞れるのが最大の強み。**

```
【例】1着: 1号艇 / 2着: 2,3,4 / 3着: 2,3,4,5
→ 1-2-3, 1-2-4, 1-2-5, 1-3-2, 1-3-4, 1-3-5,
  1-4-2, 1-4-3, 1-4-5 = 9点
```

### ボックス買い ⚠️

**選んだ艇の全順列を買う方法。点数が膨らみやすい。**

```
【例】1,2,3,4号艇の4艇ボックス
→ 4×3×2 = 24点
```

| 買い方 | メリット | デメリット |
|-------|---------|----------|
| フォーメーション | 点数を絞れる | 展開予想が必要 |
| ボックス | 考える量が少ない | 点数が多くトリガミになりやすい |

**結論：フォーメーションを基本にしましょう。** ボックスは点数が増えすぎて、配当を食いつぶします。

## 点数別の回収率 — 何点買いがベスト？

買い目の点数は回収率に直結します。

| 点数 | 的中率目安 | 必要オッズ | 回収率の傾向 |
|-----|----------|----------|------------|
| 6点 | 15〜20% | 30倍以上 | ⭐ 高回収率を狙える |
| 12点 | 25〜35% | 35倍以上 | ✅ バランス型 |
| 24点 | 40〜50% | 50倍以上 | ⚠️ トリガミ注意 |
| 36点以上 | 50%超 | 70倍以上 | ❌ ほぼトリガミ |

**おすすめは6〜12点**。これ以上増やすと的中しても利益が出にくくなります。

## 1号艇軸 vs 外枠軸 — 戦略の使い分け

### 1号艇軸（イン逃げ狙い）

ボートレースの1号艇勝率は全国平均で約55%。**最も安定した軸**です。

```
【鉄板パターン】
1-2-3: 最も出現率が高い
1-2-4: 4号艇差しが絡むパターン
1-3-2: 3号艇先行のパターン
```

**向いているレース：**
- イン勝率が高い会場（大村・芦屋・徳山）
- 1号艇にA1級選手＋好モーター
- 風が弱く水面が穏やか

### 外枠軸（まくり・差し狙い）

外枠を軸にすると**高配当**を狙えます。

```
【外枠軸パターン】
4-5-6: 外まくり展開
3-4-1: まくり＋イン粘り
4-1-5: 差し展開
```

**向いているレース：**
- 1号艇のスタートが遅い（展示タイミング0.20以上）
- 外枠にA1級＋モーター好調の選手
- まくりが多い会場（戸田・平和島・江戸川）

## 資金配分の考え方

### 均等買いは損をする

全買い目を同じ金額で買うのは**最も効率が悪い**方法です。

### おすすめ：傾斜配分

**本命目に厚く、押さえ目は薄く。**

```
【12点買いの資金配分例（合計2,000円）】
本命（4点）: 各300円 = 1,200円
対抗（4点）: 各100円 = 400円
押さえ（4点）: 各100円 = 400円
```

### 配分の鉄則

1. **1レースの上限は資金の5〜10%**
2. **本命目で的中しても利益が出るオッズか確認**
3. **トリガミになる買い目は削る**

## BoatAIの展開予測を活用した3連単の組み立て方

BoatAIでは、選手・モーター・展示・コース別データを統合した**AI予測**を無料で公開しています。3連単の組み立てに活用する方法を紹介します。

### 活用ステップ

1. **BoatAIで対象レースの予測を確認**
   - 1着予測の信頼度が高いレースを選ぶ

2. **1マーク展開予測をチェック**
   - 「逃げ」予測なら → イン軸フォーメーション
   - 「まくり」予測なら → 外枠軸フォーメーション

3. **荒れ度スコアで買い方を変える**
   - 荒れ度「低」→ 本命寄りの6〜8点買い
   - 荒れ度「高」→ 手広く12点買い or 外枠軸に切り替え

4. **超展開データで2・3着候補を絞る**
   - 攻撃力が高い選手 → 2着候補に
   - 守備力が高い選手 → 3着候補に

### 実践例

```
【BoatAI予測】
1着: 1号艇（信頼度85%）
展開予測: 逃げ
荒れ度: 低

【買い目】
1-2-3: 400円（本命）
1-2-4: 300円
1-3-2: 300円
1-3-4: 200円
1-4-2: 200円
1-4-3: 100円
= 6点 合計1,500円
```
