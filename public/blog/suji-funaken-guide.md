# スジ舟券とは？理論と実践で回収率を上げる買い方

## はじめに

「1-2-3」「4-5-6」という買い目を見て、**なぜこの組み合わせなのか**説明できますか？

ボートレースには「スジ」と呼ばれる**レース展開のセオリー**があります。スジを理解すれば、ただの数字の組み合わせではなく、**レース展開を想像しながら買い目を組み立てる**ことができるようになります。

この記事では、スジ舟券の基本から実践的な活用法まで徹底解説します。

---

## スジ舟券とは？

### 定義

**スジ舟券**とは、ボートレースの展開セオリーに基づいた買い目のことです。

ボートレースでは、1周目第1ターンマーク（1マーク）の攻防でほぼ勝負が決まります。このとき、**どの艇が先頭を取るかによって、後続の順位パターンが決まりやすい**という法則があります。

### なぜスジが生まれるのか？

ボートレースのコースは左回り。1マークを回るとき：

- **内側の艇**：回り足が短く、先にターンできる
- **外側の艇**：回り足が長く、遅れやすい

このため、**勝った艇のコースによって、2着・3着に来やすい艇が決まる**のです。

## 基本のスジパターン

### イン逃げのスジ（1号艇が勝つとき）

**最も基本的なパターン。1号艇がスタートを決めて逃げ切るケース。**

| 展開 | スジ目 | 理由 |
|------|--------|------|
| 1号艇逃げ | 1-2、1-3 | 内側の2,3号艇がそのまま続く |
| 1号艇逃げ（差し） | 1-4、1-2-4 | 4号艇が内を差して2着に |

**代表的な買い目：**
```
1-2-3（本命中の本命）
1-2-4（差しが入るパターン）
1-3-2（3号艇が2号艇より先行）
1-3-4
```

**ポイント：**
- 1号艇が逃げると、外の5,6号艇は基本的に来ない
- 「1-2-6」「1-5-6」は**逆スジ（ありえない目）**

---

### まくりのスジ（外枠が勝つとき）

**3〜6号艇が外から一気に攻めて1マークを先取するケース。**

#### 3号艇まくり
| 展開 | スジ目 | 理由 |
|------|--------|------|
| 3号艇まくり | 3-4、3-5 | まくった勢いで外の艇が続く |
| 3号艇まくり差し | 3-1、3-4-1 | 内を差して3号艇が出る |

**代表的な買い目：**
```
3-4-5（まくり展開）
3-4-1（外まくり＋イン粘り）
3-1-4（差し展開）
```

#### 4号艇まくり
| 展開 | スジ目 | 理由 |
|------|--------|------|
| 4号艇まくり | 4-5、4-6 | 一緒に外を回った艇が続く |
| 4号艇まくり | 4-5-6 | 完全な外まくり展開 |

**代表的な買い目：**
```
4-5-6（大外まくり）
4-5-1（まくり＋イン粘り）
4-3-5（3号艇が2着に粘る）
```

---

### 差しのスジ

**インの艇が先マイ（先にターン）するが、内側を差されるケース。**

| 展開 | スジ目 | 理由 |
|------|--------|------|
| 2号艇差し | 2-1、2-3 | 1号艇の内を差す |
| 4号艇差し | 4-1、4-2 | まくりに見せて内を差す |

**ポイント：**
- 差しは**ターンが上手い選手**の得意技
- 差しが決まると**インが2着に残りやすい**

---

## スジ早見表

覚えておくと便利な**スジ早見表**です。

| 1着 | 2着候補（スジ） | 3着候補 |
|-----|----------------|---------|
| 1号艇 | 2, 3, 4 | 2, 3, 4, 5 |
| 2号艇 | 1, 3 | 1, 3, 4 |
| 3号艇 | 1, 4, 5 | 1, 2, 4, 5 |
| 4号艇 | 3, 5, 6 | 1, 3, 5, 6 |
| 5号艇 | 4, 6 | 1, 4, 6 |
| 6号艇 | 4, 5 | 1, 4, 5 |

**法則：**
- 内枠が勝つ → **内の艇が続きやすい**
- 外枠が勝つ → **外の艇が続きやすい**

---

## 逆スジ（来にくい目）

スジを理解すると、**来にくい目**も分かります。

### イン逃げなのに外が来る目（逆スジ）
```
❌ 1-5-6（1号艇逃げで5,6が2,3着は稀）
❌ 1-6-5
❌ 1-6-2
```

### 外まくりなのにインが来る目（逆スジ）
```
❌ 4-1-2（4号艇まくりで1,2が続くのは稀）
❌ 5-1-2
❌ 6-1-2
```

**逆スジが来るとき：**
- 大波乱のレース
- 転覆・落水があった
- スタートが大きくばらけた

逆スジは**配当が高い**ですが、狙って当てるのは難しいです。

## スジを活用した買い方

### 戦略1: スジで絞る

**買い目を絞るときにスジを使う**

例：1号艇が勝ちそうなレース
```
全通り: 1-234-234 = 6点
スジで絞る: 1-2-3, 1-2-4, 1-3-2, 1-3-4 = 4点
```

2点減らせるだけで、回収率が上がります。

---

### 戦略2: 展開予想からスジを選ぶ

**スタート展示を見て展開を予想し、スジを決める**

```
【展開予想】
・1号艇のスタートが遅い
・4号艇がスタート早く、まくりそう
→ 4号艇まくりのスジを採用

【買い目】
4-5-6（まくり展開）
4-5-1（イン粘り）
4-3-5（3号艇残り）
```

---

### 戦略3: 本線と押さえを分ける

**スジ目を本線、逆スジを押さえに**

```
【本線（スジ目）】80%の資金
1-2-3: 1,000円
1-2-4: 800円
1-3-2: 600円

【押さえ（逆スジ）】20%の資金
1-5-4: 300円
1-6-4: 300円
```

## 会場別のスジ傾向

会場によってスジの傾向が異なります。

### イン逃げスジが決まりやすい場
| 会場 | 1号艇勝率 | 特徴 |
|------|----------|------|
| 大村 | 約60% | 日本一イン強い |
| 芦屋 | 約58% | 静水面でイン有利 |
| 徳山 | 約57% | 穏やかな水面 |

→ **1-2-3、1-2-4のスジが決まりやすい**

### まくりスジが決まりやすい場
| 会場 | 特徴 |
|------|------|
| 戸田 | 狭い水面でまくり有利 |
| 平和島 | 潮の影響でイン不利 |
| 江戸川 | 難水面でイン受難 |

→ **3-4-5、4-5-6のスジが出やすい**

## AIとスジの組み合わせ

BoatAIの予測とスジを組み合わせると、さらに精度が上がります。

### 使い方

1. **AIの予測を確認**
   - 1着予測: 4号艇
   - 2着予測: 5号艇
   - 3着予測: 1号艇

2. **スジに当てはめる**
   - 4号艇が1着 → 4号艇まくりのスジ
   - 4-5 はスジ通り ✓
   - 4-5-1 は「まくり＋イン粘り」で自然 ✓

3. **スジ目なら信頼度UP**
   - AIの予測がスジに合致 → 信頼度高
   - AIの予測が逆スジ → 注意が必要

## よくある質問

### Q. スジ通りに買えば勝てる？

**A. スジだけでは勝てません。**

スジはあくまで「来やすいパターン」であり、100%ではありません。重要なのは：
- スタート展示での展開予想
- 選手の実力とモーター性能
- 風や波の影響

スジは**買い目を絞るための道具**として使いましょう。

### Q. 逆スジは買わない方がいい？

**A. 状況によります。**

- 普段は**スジ目を優先**
- 荒れそうなレースでは**逆スジも少額で押さえる**
- 逆スジが来たときの配当は高いので、保険として有効

### Q. スジはどうやって覚える？

**A. 「勝った艇の近くが来る」と覚えましょう。**

- 1号艇が勝つ → 2,3号艇が続く（近い）
- 4号艇が勝つ → 5,6号艇が続く（近い）
- 例外は「差し」で内が残るパターン

## まとめ

### スジ舟券のポイント

1. **スジとは展開のセオリー**
   - 内が勝てば内が続く
   - 外が勝てば外が続く

2. **スジで買い目を絞る**
   - 全通り買いは資金効率が悪い
   - スジで2-3点絞れる

3. **逆スジは来にくい**
   - 「1-5-6」「4-1-2」は稀
   - 来たら高配当だが狙いにくい

4. **会場特性を考慮**
   - イン強い場 → イン逃げスジ
   - まくり多い場 → まくりスジ

5. **AIとの併用が効果的**
   - AIの予測がスジに合致 → 信頼度UP
   - スジ外れの予測 → 慎重に
