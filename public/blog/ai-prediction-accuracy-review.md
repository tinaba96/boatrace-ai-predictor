# ボートレースAI予想は本当に当たるのか？ — 3ヶ月15,000レースの検証結果

「AI予想って、本当に当たるの？」

ボートレースのAI予想サービスが増えてきた今、こう感じている方は多いでしょう。正直に言えば、**疑うのが正しい姿勢**です。

「的中率90%以上！」「毎月100万円の利益！」といった誇大広告が横行するこの業界で、数字を鵜呑みにするのは危険です。

## AI予想への懐疑 — 疑うべき3つの理由

AI予想に対する不信感には、根拠があります。

- **実績を公開しないサービスが多い** — 当たったときだけスクリーンショットを載せ、外れたレースは無視する
- **「AI」という言葉がマーケティングに利用されている** — 実態はExcelの単純計算でも「AI搭載」と謳えてしまう
- **検証期間やサンプル数が不明** — 1週間だけの好成績を「実績」として掲載する

BoatAIはこの問題に正面から向き合うため、**予想した全レースの結果をサイト上で公開**しています。当たったレースも外れたレースも、誰でも確認できます。

## 3ヶ月・15,000レースの検証結果

### 全体の的中率

| 指標 | 数値 |
|------|------|
| 検証レース数 | **約15,000レース** |
| 検証期間 | 3ヶ月間 |
| 単勝的中率（本命が1着） | **約47〜50%** |
| 複勝的中率（本命が2着以内） | **約60〜65%** |
| 3連複的中率 | **約18%** |

ボートレースは6艇で争われるため、ランダムに選んだ場合の単勝的中率は約16.7%です。**AIの47〜50%という数字は、ランダムの約3倍**の精度です。

一方で、「半分は外れる」という事実も見逃せません。AI予想は万能ではなく、**約半数のレースでは予想が外れます**。

### 3連複的中率18%の意味

3連複は3着以内の3艇を当てる必要があり、全組み合わせは20通り。ランダムの的中率は5%です。

18%という数字は**ランダムの3.6倍**ですが、逆に言えば**5回に4回以上は外れる**ということ。3連複を毎レース買い続けて勝つのは簡単ではありません。

## 回収率の現実 — 当たるけど儲かるのか？

ここが最も正直に話すべきポイントです。

### 全レース均一購入では回収率100%は難しい

ボートレースの控除率は約25%。つまり、賭け金全体のうち約25%は最初から差し引かれています。

全レースに均一金額で賭け続けた場合、BoatAIの通常予想の回収率は**約88〜94%**です。的中率はランダムの3倍ありますが、それでも控除率の壁を完全に超えるのは容易ではありません。

### 「今日のおすすめ」で回収率104%を達成

BoatAIには「**今日のおすすめ**」という機能があります。AIが自信度の高いレースだけを厳選してピックアップする機能です。

| 指標 | 通常予想 | 今日のおすすめ |
|------|----------|---------------|
| 対象レース数 | 全レース | 厳選レースのみ |
| 回収率 | 88〜94% | **104%** |
| 的中率 | 18%（3連複） | **30%**（3連複） |

つまり、**「全レースをAI任せで買う」のではなく「AIが厳選したレースだけを買う」**ことで、控除率の壁を突破できる可能性があるということです。

ただし、この数字も将来を保証するものではありません。市場環境やオッズの変動により、成績は常に変化します。

## AI予想の限界と正しい使い方

### AIにできないこと

どれだけ精度が高くても、AIには限界があります。

- **レース直前の体調変化やトラブルは検知できない** — 選手の当日コンディションまではデータに反映されない
- **スタートタイミングの予測は完璧ではない** — フライングや出遅れは予測困難
- **オッズの変動を完全に予測できない** — 締切直前の大口投票で期待値が変わる
- **100%の的中は不可能** — どんなに優れたAIでも外れるレースは必ずある

### AI予想の正しい使い方

1. **判断材料の一つとして使う** — AIの予想を鵜呑みにせず、自分でもレース情報を確認する
2. **レースを選ぶ** — 全レース購入ではなく、自信度の高いレースに絞る
3. **資金管理を徹底する** — 1日の予算を決め、一発逆転を狙わない
4. **長期的な視点を持つ** — 1日の勝ち負けではなく、1ヶ月単位で成績を評価する
5. **モデルを使い分ける** — 堅いレースには本命モデル、荒れそうなレースには穴モデル
