# モーター抽選日から勝負が始まる — モーター2連率の正しい読み方と落とし穴

「モーター2連率が高い選手を買っておけば間違いない」

そう思っていませんか？ 実は、モーター2連率には**正しく読まないと損をする落とし穴**がいくつもあります。

この記事では、モーター2連率の仕組みから部品交換の見方、展示タイムとの組み合わせ評価まで、モーターを深く理解するための知識を解説します。

---

## モーター2連率とは — 基本をおさらい

### 2連率の定義

モーター2連率とは、そのモーターを使用した全レースのうち**1着または2着に入った割合**です。

```
2連率 = （1着回数 + 2着回数）÷ 出走回数 × 100
```

### 重要：節ごとにリセットされない

モーターの2連率は**約1年間の累積データ**です。各ボートレース場では年に1回モーターが新品に入れ替わり、そこからデータが蓄積されていきます。

| 項目 | 内容 |
|-----|------|
| 更新タイミング | 毎レース後にリアルタイム更新 |
| リセット | 年1回のモーター入れ替え時 |
| 使用期間 | 約1年間 |
| 基準値 | **約30%が平均ライン** |

---

## 2連率の正しい読み方 — 3つの注意点

### 注意点1: 節序盤のデータは信頼できない

モーターが入れ替わった直後の1〜2ヶ月は、**使用回数が少なくデータが安定しません**。

| 使用回数 | 信頼度 | 判断 |
|---------|--------|------|
| 30回未満 | **低い** | 参考程度にとどめる |
| 30〜60回 | やや信頼できる | 傾向が見えてくる |
| 60回以上 | **高い** | 信頼して判断できる |

使用回数が少ないときは、2連率の数値よりも**展示タイムや展示航走の目視**を重視しましょう。

### 注意点2: 30%が基準ライン

6艇で走るボートレースでは、均等に結果が出れば2連率は約33%になります。実際には選手の実力差があるため、**30%前後が平均的なモーター**と考えてください。

| 2連率 | 評価 |
|-------|------|
| **40%以上** | エースモーター。積極的に狙える |
| **35〜40%** | 優良モーター。選手次第で本命候補 |
| **30〜35%** | 平均的。他の要素で判断 |
| **25〜30%** | やや低調。警戒が必要 |
| **25%未満** | ハズレモーター。割引いて評価 |

### 注意点3: 過去の使用選手を見る

2連率は**誰が乗ったかに大きく左右**されます。

```
【例】モーター No.45（2連率38%）

直近の使用選手：
- A1級・峰竜太クラス → 2連率55%
- B2級選手 → 2連率20%
→ 平均すると38%だが、実態は選手次第
```

**A1級選手が乗って高い2連率** → モーターが良いとは限らない
**B1級選手が乗っても高い2連率** → 本物のエースモーター

## モーター交換と部品交換の重要性

### モーター交換とは

節間にモーター本体を交換するケースがあります。交換後は**過去のデータがリセット**されるため要注意です。

- 出走表に「モーター交換」の記載がないか確認
- 交換直後は過去の2連率が参考にならない
- 展示航走の走りで判断するしかない

### 部品交換の見方

選手は前検日や節間に部品を交換して整備を行います。**どの部品を交換したかで、選手の意図が読み取れます。**

| 交換部品 | 意図 | 予想への影響 |
|---------|------|------------|
| **ピストン** | 出力アップ狙い | 伸び足が改善される可能性 |
| **ピストンリング** | 出力微調整 | 軽微な調整。大きな変化は少ない |
| **キャブレタ** | 燃料供給の調整 | 回転数の安定に影響 |
| **電気系統** | 点火タイミング | スタート時の加速に影響 |
| **ギアケース** | ペラとの相性調整 | 回り足に影響 |

**複数パーツを同時交換 → モーターの状態が悪い可能性が高い。** 逆に交換なし or 軽微な交換のみなら、モーターの素性が良い証拠です。

## 伸び足型 vs 回り足型 — モーターのタイプを見極める

モーターには大きく2つのタイプがあります。

### 伸び足型

- **直線でスピードが出る**タイプ
- ダッシュスタートの外枠で有利
- まくりが決まりやすい
- **展示タイムが速い**のが特徴

### 回り足型

- **ターンがスムーズ**なタイプ
- インコースで有利
- 先行してそのまま逃げ切るパターン
- **展示タイムは普通でもターンで差をつける**

### コース別の有利不利

| モータータイプ | インコース | 外枠 |
|-------------|----------|------|
| 伸び足型 | △（ターンで詰められる） | **有利** |
| 回り足型 | **有利** | △（直線で離される） |

## 展示タイムとの組み合わせ評価

モーター2連率だけでなく、**展示タイムと組み合わせる**ことで精度が上がります。

### 4パターンの評価マトリクス

| | 展示タイム速い | 展示タイム遅い |
|---|-------------|-------------|
| **2連率高い** | 最強。本命候補 | 回り足型の可能性。インなら狙える |
| **2連率低い** | 整備で改善された可能性。穴候補 | 厳しい。割引いて評価 |

### 特に注目すべきパターン

**2連率低い + 展示タイム速い：**
これは**穴モーター**の典型パターンです。過去は不調でも、今節の選手が整備で仕上げてきた可能性があります。オッズが高くなりやすいため、期待値の高い舟券になり得ます。

**2連率高い + 展示タイム遅い：**
モーター本来の力はあるが、**今節のコンディションが合っていない**可能性。過信は禁物です。
