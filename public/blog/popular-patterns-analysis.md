# 出目買いは有効か？ — 全24場の出目データをAIが分析した結果

## はじめに

「1-2-3ばっかり来るなら、毎回それだけ買えば勝てるのでは？」

ボートレースをやっていると、一度はそう考えたことがあるはずです。特定の着順パターン（出目）を繰り返し買い続ける戦略、いわゆる**出目買い**。シンプルで分かりやすい手法ですが、本当に有効なのでしょうか？

この記事では、全24場の出目データをもとに、出目買いの実態と限界を分析します。

---

## 出目買いとは？

出目買いとは、**レースの展開や選手データを考慮せず、特定の着順パターン（出目）を毎レース機械的に買い続ける戦略**です。

**例:**
- 「1-2-3」だけを毎レース買う
- 「1号艇が1着になる出目」だけを買う
- 過去の統計で出やすい出目上位5パターンを買う

予想の手間がかからないため、初心者に人気がある手法です。

---

## 全体の出目ランキング — 最も出やすい出目は？

全24場のデータを集計すると、3連単の出目には明確な偏りがあります。

### 出目出現率ランキング（3連単・全場平均）

| 順位 | 出目 | 出現率 |
|------|------|--------|
| 1位 | 1-2-3 | 約10.5% |
| 2位 | 1-3-2 | 約6.0% |
| 3位 | 1-2-4 | 約5.5% |
| 4位 | 1-3-4 | 約4.0% |
| 5位 | 1-2-5 | 約3.5% |
| 6位 | 2-1-3 | 約3.2% |
| 7位 | 1-4-2 | 約3.0% |
| 8位 | 1-2-6 | 約2.8% |
| 9位 | 1-4-3 | 約2.5% |
| 10位 | 2-1-4 | 約2.3% |

**圧倒的に1号艇の1着が多い**のがボートレースの特徴です。上位10パターンのうち8つが1号艇1着の出目で占められています。これはインコース有利というボートレースの構造的な特性によるものです。

3連単の全通り数は120通り。完全にランダムなら1通りあたり約0.83%ですが、「1-2-3」は約10.5%と**平均の12倍以上の出現率**を誇ります。

## 会場別の出目の違い — 同じ出目でも場によって全然違う

ここが出目買いの落とし穴です。全体平均だけを見ると「1-2-3を買えばいい」と思いがちですが、**会場によって出目の傾向は大きく異なります**。

### 1号艇1着率の会場比較

| 会場 | 1号艇1着率 | 特徴 |
|------|-----------|------|
| 大村 | 約65% | 全国トップのイン天国 |
| 芦屋 | 約60% | 安定したイン成績 |
| 徳山 | 約60% | 穏やかな水面でイン有利 |
| 江戸川 | 約40% | 全国屈指の荒れ水面 |
| 戸田 | 約43% | 狭い水面でまくり多発 |
| 平和島 | 約44% | 潮の影響で波乱含み |

**大村と江戸川では1号艇の1着率に約25%もの差**があります。つまり「1-2-3」の出現率も会場によって2倍近く変わるのです。

### 会場別の注目出目

- **大村**: 「1-2-3」の出現率が約15%。出目買いが最も機能しやすい
- **江戸川**: 「1-2-3」の出現率は約6%。代わりに2号艇・3号艇の1着出目が多い
- **戸田**: まくりが決まりやすく「2-1-3」「3-1-2」などの出目が他場より高い

## 出目買いのメリット・デメリット

### メリット

- **予想の手間がゼロ** — データ分析や選手研究が不要
- **感情に左右されない** — 機械的に買うので追い上げなどの失敗が起きにくい
- **初心者でも始めやすい** — 知識がなくても実行可能

### デメリット

- **回収率が上がりにくい** — 出やすい出目ほどオッズが低い
- **レースごとの状況を無視** — 当日の選手コンディションやモーター性能が反映されない
- **長期的に資金が減る** — トリガミのリスクが常にある

---

## データで見る出目買いの回収率

出目買いの最大の問題は、**出やすい出目ほどオッズが低い**という点です。

### 出目別の平均回収率（3連単・全場平均）

| 出目 | 出現率 | 平均オッズ | 推定回収率 |
|------|--------|-----------|-----------|
| 1-2-3 | 10.5% | 6.5倍 | 約68% |
| 1-3-2 | 6.0% | 10倍 | 約60% |
| 1-2-4 | 5.5% | 12倍 | 約66% |
| 2-1-3 | 3.2% | 18倍 | 約58% |
| 4-1-2 | 1.5% | 45倍 | 約68% |

**いずれの出目も回収率が70%を下回っています。**

これはボートレースの控除率（約25%）が大きく影響しています。出目買いは「当たるかどうか」ではなく「当たった時の配当で購入額を上回れるか」が問題であり、機械的に買い続ける限り控除率の壁を超えることは困難です。

---

## 出目買いよりもデータ分析が有効な理由

出目買いが回収率70%前後に留まるのに対し、**レースごとのデータ分析を行うことで回収率を大幅に改善**できます。

### 分析すべきデータ

- **選手の実力** — 勝率、2連対率、当地成績
- **モーター性能** — モーター2連対率、展示タイム
- **コース別成績** — 選手ごとの得意コース
- **当日の気象条件** — 風向き・風速による展開変化
- **スタートタイミング** — 展示のスタート情報

出目の統計は「過去の結果の平均」に過ぎません。一方、**レースごとのデータ分析は「今日のこのレース」に最適化された予想**を導き出します。

同じ「1-2-3」でも、1号艇がA1級でモーター好調なレースと、B1級でモーター不調なレースでは、期待値がまったく異なります。この違いを見極めることが、回収率100%超への鍵です。

## まとめ

**出目買いの結論:**
1. 「1-2-3」が最も出やすいのは事実だが、回収率は約68%
2. 会場によって出目傾向は大きく異なる
3. 出目買いだけでは控除率の壁を超えられない
