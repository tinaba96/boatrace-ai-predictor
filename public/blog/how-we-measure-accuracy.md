# BoatAIの実績は本物か？計測方法と透明性を徹底解説

「的中率49.5%」「回収率134.3%」

BoatAIのトップページに表示されているこれらの数字。**本当に信じていいの？**と思う方も多いでしょう。

正直に言います。**疑って当然です。**

世の中には「的中率90%！」「月収100万円確定！」といった怪しい予想サイトが溢れています。根拠のない数字を並べて、高額な会員費を取るサイトも少なくありません。

だからこそ、BoatAIは**全ての実績データを公開**しています。

この記事では、BoatAIがどのように実績を計測しているのか、なぜ全データを公開しているのか、徹底的に解説します。

---

## なぜ実績を全公開しているのか

### 高額予想サイトへの怒りから生まれたBoatAI

BoatAI開発者の私自身、かつて月額3万円の予想サイトに登録していました。

「プロの予想」を謳いながら、的中率も回収率も非公開。当たったときだけ大々的にアピールし、外れたときは何も言わない。そんなサイトに何万円も払いました。

結果？**惨敗**でした。

その経験から、「予想を提供するなら、実績は全て公開すべきだ」という信念でBoatAIを開発しました。

### 都合の良い数字だけを切り取らない

多くの予想サイトは「当たったレースだけ」を実績として公開します。

- 「3連単10万円的中！」→ その前に20回外れていることは言わない
- 「月間回収率200%！」→ 計測期間や対象レースが不明

BoatAIは違います。**当たったレースも外れたレースも、全て公開**しています。

## 実績の計測方法

### 1. 対象となるレース

BoatAIが予想を公開した**全レース**が対象です。

- 特定のレースだけを選んで計測することはしません
- 「自信度が高いレースだけ」といった恣意的な選別もしません
- 予想を公開した時点で、そのレースは必ず集計対象になります

### 2. 的中の判定基準

#### 複勝的中率
AIが予想した「本命」の艇が**2着以内**に入れば的中としてカウントします。

```
複勝的中率 = 本命が2着以内に入ったレース数 ÷ 全レース数 × 100
```

#### 3連複的中率
AIが予想した買い目の中に、**実際の3着以内の3艇の組み合わせ**が含まれていれば的中です。

```
3連複的中率 = 3連複が当たったレース数 ÷ 全レース数 × 100
```

#### 3連単的中率
AIが予想した買い目の中に、**実際の1着-2着-3着の順番通りの組み合わせ**が含まれていれば的中です。

```
3連単的中率 = 3連単が当たったレース数 ÷ 全レース数 × 100
```

### 3. 回収率の計算方法

回収率は「投資した金額に対して、どれだけ戻ってきたか」を示す指標です。

```
回収率 = 払戻金の合計 ÷ 投資金額の合計 × 100
```

#### 具体例

| レース | 投資額 | 結果 | 払戻金 |
|--------|--------|------|--------|
| 1R | 1,000円 | 的中 | 3,500円 |
| 2R | 1,000円 | 不的中 | 0円 |
| 3R | 1,000円 | 不的中 | 0円 |
| 4R | 1,000円 | 的中 | 2,200円 |
| 5R | 1,000円 | 不的中 | 0円 |
| **合計** | **5,000円** | - | **5,700円** |

この場合の回収率：
```
5,700円 ÷ 5,000円 × 100 = 114%
```

**回収率100%以上**であれば、投資した金額以上が戻ってきている（＝プラス収支）ということです。

### 4. 投資額の前提

BoatAIの回収率計算では、**1点あたり100円**を想定しています。

例えば、3連単の推奨買い目が10点の場合：
- 投資額：100円 × 10点 = 1,000円
- 払戻金：的中時のオッズ × 100円

この前提を統一することで、レース間の比較が可能になります。

## 各指標の意味と目安

### 的中率と回収率、どっちが大事？

**結論：回収率の方が重要**です。

的中率が高くても、回収率が低ければ意味がありません。

| パターン | 的中率 | 回収率 | 結果 |
|----------|--------|--------|------|
| A | 50% | 80% | マイナス |
| B | 30% | 120% | **プラス** |
| C | 10% | 150% | **プラス** |

パターンAは半分当たっていますが、オッズが低すぎて回収率80%。つまり、1万円投資しても8,000円しか戻ってきません。

一方、パターンCは的中率10%ですが、回収率150%。1万円投資すれば15,000円になります。

### BoatAIが目指す数値

| 指標 | 目標 | 現在の実績 |
|------|------|-----------|
| 複勝的中率 | 45%以上 | 49.5% |
| 複勝回収率 | 80%以上 | 85.3% |
| 3連複的中率 | 10%以上 | 12.8% |
| 3連単回収率 | 100%以上 | 134.3% |

特に**3連単回収率100%以上**を維持することを最重要目標としています。

## なぜBoatAIの数字を信頼できるのか

### 1. 全レースのデータを公開

BoatAIの「成績」ページでは、日別の的中率・回収率を全て確認できます。

- 良かった日も悪かった日も隠さず公開
- 各モデル（スタンダード・本命狙い・穴狙い）別の成績も公開
- 過去の予想と結果も「過去の予想」ページで確認可能

### 2. 予想は事前に公開

BoatAIの予想は、**レース開始前**に公開されています。

- 後出しで「実は当たってました」は不可能
- 公開時刻も記録されている
- 誰でもリアルタイムで予想を確認できる

### 3. 結果は自動で集計

レース結果が出ると、システムが自動で的中判定と集計を行います。

- 人間の手で数字を操作する余地がない
- オッズも公式データを使用
- 計算ミスや意図的な改ざんを防止

### 4. コードはオープン（予定）

将来的には、集計ロジックのコードをGitHubで公開する予定です。

「本当にこの計算方法で集計しているのか？」という疑問にも、コードを見れば答えが分かります。

## よくある質問

### Q. 回収率134%って本当？

**A. 本当です。**ただし、これは2025年12月時点の数字です。

ボートレースは「運」の要素もあるため、短期的には数字が上下します。長期的に100%以上を維持することを目標としています。

### Q. 過去の悪い実績を消していない？

**A. 消していません。**「成績」ページで過去の日別データを全て確認できます。

回収率50%を下回った日も、包み隠さず公開しています。

### Q. 自分で検証できる？

**A. できます。**

1. BoatAIの予想を毎日確認
2. レース結果と照らし合わせる
3. 自分で的中率・回収率を計算

実際に検証している方もいらっしゃいます。疑わしいと思ったら、ぜひご自身で確認してください。

## まとめ

BoatAIの実績は、以下の原則で計測・公開しています。

1. **全レースを対象**：都合の良いレースだけを選ばない
2. **当たりも外れも公開**：的中したレースだけをアピールしない
3. **計測方法を明示**：どうやって数字を出しているか説明する
4. **自動集計**：人間の手で操作できない仕組み
5. **誰でも検証可能**：予想も結果も全て公開

「無料だから適当」ではなく、「無料でも本物」を証明し続けます。

BoatAIの実績に疑問があれば、ぜひ実際に検証してみてください。そして、納得いただけたら、ぜひ予想を活用してください。
