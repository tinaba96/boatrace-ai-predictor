# 【2026年4月】BoatAI 月間実績レポート - 3モデル×4,278レースの全成績公開

## はじめに

2026年4月が終わりました。BoatAIは4月だけで**4,278レース**を分析しました。

4月は改善施策が本格化した月です。会場別1コース勝率の動的化、イン崩れ指数の会場別重みチューニング、AIデータ予想テーブルへの◎○△推奨記号追加など、複数の機能改善を実施しました。

この記事では、4月の全成績を**包み隠さず**公開します。

**この記事でわかること**:
- 3モデルの的中率・回収率の比較（単勝・複勝・3連単・3連複）
- 3月との成績比較と傾向分析
- 4月に実施した主要な改善
- 会場別の好成績・課題分析

## モデル別成績

### 本命狙いモデル（safeBet）

**特徴**: 人気上位の選手が有利な堅いレースに特化。3モデル中で最も的中率が高い。

| 舟券種別 | 的中率 | 回収率 |
|---------|--------|--------|
| **単勝** | **55.4%** | **88.1%** |
| **複勝** | **71.9%** | **90.8%** |
| 3連複 | 20.5% | 80.2% |
| 3連単 | 8.3% | 82.3% |

**分析**:
- **単勝的中率55.4%は驚異的な水準** — 約2回に1回は本命が的中
- **複勝回収率90.8%**は3月（95.6%）から若干低下したが、依然として堅実
- 3連複・3連単は的中数が少なく、この両舟券には最適化していない傾向

---

### スタンダードモデル

**特徴**: 全レースを対象に、45項目を総合分析。最もバランスの取れたモデル。

| 舟券種別 | 的中率 | 回収率 |
|---------|--------|--------|
| **単勝** | **16.8%** | **68.5%** |
| **複勝** | **43.0%** | **87.8%** |
| 3連複 | 18.1% | 85.7% |
| 3連単 | 2.7% | 86.9% |

**分析**:
- 単勝的中率16.8%は3月（26.8%）から低下。これは**イン崩れ指数の会場別重みチューニング**の副作用
- 複勝回収率87.8%は堅実。複勝メインでの運用が効果的
- **3連単回収率86.9%**は着順予測の精度が高い証拠

---

### 穴狙いモデル（upsetFocus）

**特徴**: 人気薄の選手が好走するレースを狙い撃ち。ハイリスク・ハイリターン型。

| 舟券種別 | 的中率 | 回収率 |
|---------|--------|--------|
| 単勝 | 14.8% | 86.9% |
| **複勝** | **35.4%** | **89.4%** |
| 3連複 | 8.6% | 46.9% |
| 3連単 | 1.8% | 58.3% |

**分析**:
- **複勝35.4%は穴狙いとしては上々の的中率**
- 3連複・3連単は的中率が低く、全レース適用には向かない
- 荒れ度が高いレース（65以上）に限定して活用することで効果的

## 4月に実施した主要な改善

### 1. イン崩れ指数の会場別重みチューニング（BOA-85）

各会場の逃げ決まり率が異なることに着目。18会場それぞれについて、イン崩れ予測の重みをデータから自動計算し直しました。

**効果**:
- イン崩れ予測精度が向上
- 荒れ度スコアがより正確に
- 但し、スタンダードモデルの単勝的中率が低下（トレードオフ）

### 2. 会場別1コース勝率の動的化（BOA-88）

従来は全国平均（約60%）で固定していた「1コース勝率」を、**直近90日の実績値**に変更。

```
表示例: 徳山では1コース勝率 67.3%（直近90日実績）
```

**効果**:
- ユーザーが会場の特性を正確に理解可能に
- 購買判断の根拠がより信頼性高く

### 3. AIデータ予想テーブルの列内ランク表示を改善（BOA-78）

各統計列（選手勝率、モーター性能など）で、該当艇を列内ランキングで表示。最高評価の数値は太字・色付けで強調。

```
例：選手勝率の列で「45.8%」が最高評価なら、その数値を太字・カラー表示
```

**効果**:
- モバイル画面でも一目で注目ポイントがわかる
- ユーザビリティが向上
- シンプルで視認性の高い表現

### 4. RaceCard にイン崩れ指数バッジを追加（BOA-75）

トップページのレースカード上に、イン崩れ予測バッジを表示。

```
🌪️ イン崩れ確率高 / 🎯 本命有利
```

**効果**:
- ユーザーが一覧画面で荒れレースを即座に識別可能
- 戦略的なレース選択をサポート

### 5. イン崩れ確率高レースの結果セクション表示（BOA-82）

RaceDetail ページで、イン崩れ予測レースの実際の結果（1コース飛び：有/無）を可視化。

**効果**:
- 予測精度の検証がユーザーに見やすく
- 信頼性向上

## 正直に振り返る：4月の課題

### スタンダードモデルの単勝的中率低下

4月の会場別重みチューニングにより、スタンダードモデルの単勝的中率が低下しました（26.8% → 16.8%）。

これは**意図的なトレードオフ**です。イン崩れ予測の精度を上げるため、スタンダードモデルは最も確率が高いパターンではなく、2番目に高いパターンを参照する設計となっています。

**ユーザーへの推奨**: 堅い予想をしたい場合は**本命狙いモデルを使用**してください。本命狙いモデルは最高確度のパターンを参照しており、4月の単勝的中率は**55.4%**です。

### 複勝回収率の微減

本命狙いモデルの複勝回収率が3月の95.6%から4月の90.8%に低下。安定性は保たれていますが、100%超えへの道がまだ遠い状況です。

### 1. 多摩川・常滑のヘビーローテーション

4月に回収率136.9%（多摩川）、125.9%（常滑）を達成した会場。5月もこれらの会場を重点的にフォローする価値がある。

### 2. 本命狙いモデルの複勝を軸に

- **複勝回収率90.8%**は全モデル中で最も100%に近い
- 会場を絞ることで100%超えを狙う戦略が効果的

### 3. イン崩れ指数の動的性能追跡

4月に導入した「会場別重み」が実環境でどう機能するかを継続監視。5月のデータで微調整を検討。

### 4. スタンダードモデルは複勝メイン

単勝は本命狙いに任せ、スタンダードは**複勝（回収率87.8%）**と**3連単（回収率86.9%）**での活用を推奨。

## データの透明性について

### 計算方法

- **的中率**: 的中レース数 / 全対象レース数 × 100
- **回収率**: 払戻額 / 投資額 × 100（1レース100円均一賭けで計算）
- **複勝の判定**: 本命が2着以内に入ったら的中
- **3連単の判定**: 上位3艇の着順が完全一致したら的中
- **3連複の判定**: 上位3艇の組み合わせが一致したら的中（着順不問）

### いつでも確認可能

このレポートの数値は、BoatAIの[成績ページ](/accuracy)で公開しているデータと一致しています。**良い数字も悪い数字も隠しません。**
