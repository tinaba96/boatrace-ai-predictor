# 進入予想のコツ - 枠なり崩れを見抜く方法

## はじめに

「枠番通りに進入すると思ったのに、全然違った...」

ボートレースでは、**枠番と実際のコースが異なる**ことがあります。これを予測できるかどうかで、舟券の精度は大きく変わります。

この記事では、進入予想のコツと、枠なり崩れを見抜く方法を解説します。

---

## 進入とは？

### 基本用語の確認

**枠番:**
抽選で決まる番号。1〜6号艇。

**コース:**
実際にスタートする位置。1〜6コース。

**枠なり:**
枠番通りにコースが決まること。
- 1号艇 → 1コース
- 2号艇 → 2コース
- ...

**枠なり崩れ:**
枠番とコースが異なること。

### なぜコースが変わるのか？

ボートレースでは、**ピットアウトからスタートまでの間に、選手が自由にコースを選べます**。

**コース取りの流れ:**
1. ピットアウト（待機行動開始）
2. 選手が狙うコースに向かう
3. 早く内側に入った選手が内コースを取れる
4. コースが確定してスタート

**つまり:**
内側のコースを取りたい選手は、早めに動いて内側に入ろうとします。これが「前付け」です。

## 進入パターンの種類

### パターン1: 枠なり

最も一般的なパターン。特に近年は枠なりが増えています。

```
枠番: 1-2-3-4-5-6
進入: 1-2-3-4-5-6
```

**枠なりになりやすい条件:**
- 1号艇に強い選手がいる
- 前付け傾向のある選手がいない
- 一般戦（グレードレースは動きやすい）

### パターン2: 前付け

外枠の選手が内側のコースを取りに行くパターン。

```
枠番: 1-2-3-4-5-6
進入: 1-4-2-3-5-6（4号艇が2コースへ前付け）
```

**影響:**
- 前付けした選手は助走距離が短くなる
- 押し出された選手はスタートが難しくなる
- レースが荒れやすい

### パターン3: 深イン

1コースの選手がスタート位置を下げるパターン。

```
通常: 1コースはスタートラインに近い位置
深イン: 1コースがスタートラインから離れた位置
```

**原因:**
- 前付けに対抗してコースを守ろうとした
- スタート位置が後ろになってしまった

**影響:**
- 1号艇の助走距離が長くなる
- スタートで遅れるリスク
- 逃げが難しくなる

### パターン4: 回り直し

一度コースを取った後、やり直すパターン。

**原因:**
- 待機行動違反を避けるため
- 進入が思い通りにいかなかった

**影響:**
- 助走距離が極端に短くなる
- スタートが非常に難しくなる

## 前付け選手を見抜く方法

### 方法1: 選手の「平均ST」と「進入コース」を確認

**前付け傾向のある選手の特徴:**
- 平均STが0.18以上（スロースタート）
- 枠番より内側のコースに入ることが多い
- 進入データで「イン率」が高い

**確認方法:**
出走表の「進入コース」欄を見る。過去のコース取り傾向が分かります。

### 方法2: ベテラン選手に注目

**前付けが多い選手の傾向:**
- ベテラン選手（40歳以上）
- 峰竜太、白井英治など一部のトップ選手
- スタートに自信がない選手

**理由:**
若手選手は枠なりでスタート勝負する傾向がありますが、ベテラン選手はコース取りで勝負することがあります。

### 方法3: 出走表の「F/L」を確認

**F（フライング）やL（出遅れ）持ちの選手:**
- 安全にスタートしたい心理
- 前付けして助走距離を短くすることも
- または、枠なりで確実にスタートすることも

**判断が難しい:**
F/L持ちの選手の行動は読みにくいため、要注意。

## 枠なり崩れの影響を予測する

### 影響1: 1号艇が深インになった場合

**想定されるレース展開:**
- 1号艇のスタートが遅れる可能性
- 2〜3コースの差し・まくり差しが決まりやすい
- 1号艇が逃げ切れないことも

**舟券の組み立て:**
- 1号艇を1着から外す
- 2〜3号艇を1着候補に
- 荒れることを想定して手広く

### 影響2: 外枠選手が前付けした場合

**想定されるレース展開:**
- 前付け選手は助走不足でスタートが遅れやすい
- 押し出された選手はスタートしにくい
- ダッシュ勢（4〜6コース）が有利になることも

**舟券の組み立て:**
- 前付け選手は2〜3着付け
- 外枠から攻める選手を狙う
- 混戦を想定

### 影響3: 進入が大きく変わった場合

**例: 5号艇が1コースに入った場合**

```
枠番: 1-2-3-4-5-6
進入: 5-1-2-3-4-6
```

**想定されるレース展開:**
- 全員の助走距離が変わる
- セオリーが通用しない
- 大荒れの可能性

**舟券の組み立て:**
- 無理に買わない（見送り）
- 買うなら高配当狙いでワイドや3連複

## 進入予想の実践テクニック

### テクニック1: 展示で進入を確認する

**スタート展示で見るべきポイント:**
- 実際のコース取り
- 深インになっていないか
- 前付けの動きがあるか

**注意:**
展示と本番で進入が変わることもあります。あくまで参考程度に。

### テクニック2: 選手コメントを確認する

**前検日や出走表のコメント:**
- 「インから勝負」→ 枠なりの可能性
- 「スタートを決めて」→ ダッシュ勝負の可能性
- 「コース取りから」→ 前付けの可能性

### テクニック3: 会場の傾向を把握する

**会場によって進入傾向が異なります:**
- **枠なりが多い場**: 住之江、若松など
- **前付けが起きやすい場**: 江戸川、平和島など

**理由:**
水面の広さやピットの位置によって、コース取りのしやすさが変わります。

---

## 進入予想が難しい場面

### 難しい場面1: 全員が前付け傾向

複数の選手に前付け傾向がある場合、誰がどのコースに入るか予測が困難です。

**対策:**
- 展示を必ず確認
- 買い目を絞りすぎない

### 難しい場面2: 初日のレース

選手のコンディションや戦略が分からない初日は、進入予想が難しくなります。

**対策:**
- 初日は見送りor少額
- 2日目以降にデータを元に予想

### 難しい場面3: グレードレース

SGやG1では、普段と違う動きをする選手がいます。

**対策:**
- 得点率を確認（勝負駆けの可能性）
- 選手の過去のグレード実績を確認

---

## まとめ

**進入予想のポイント:**

1. **前付け選手を見抜く**: 平均ST、進入データ、ベテラン選手に注目
2. **展示で確認**: スタート展示の進入を必ずチェック
3. **影響を予測**: 深インや前付けがあれば、展開予想を修正
4. **会場の傾向**: 枠なりが多い場、動きやすい場を把握

進入予想ができるようになると、舟券の精度は格段に上がります。

最初は難しく感じるかもしれませんが、レースを見続けることで徐々に分かるようになります。ぜひ実践してみてください。
