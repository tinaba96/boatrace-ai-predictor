# 「今日のおすすめ」が回収率104%を達成 - 2,577レースの実績データ

## はじめに

「ボートレースで回収率100%を超えるのは無理」

そう思っていませんか？

BoatAIの「**今日のおすすめ**」機能は、運用開始から約1ヶ月半で**累計回収率104%**を記録しています。これは投資額に対して**4%のプラス収支**を意味します。

この記事では、2,577レース分の実績データを公開し、この数字がどれだけ意味のあるものかを解説します。

---

| 項目 | 数値 |
|------|------|
| 運用期間 | 2026年1月16日〜 |
| 対象レース数 | **2,577レース** |
| 的中数 | **772件** |
| 的中率 | **30%** |
| 投資額（1レース100円） | 257,700円 |
| 回収額 | **269,170円** |
| **回収率** | **104%（+11,470円）** |

これは、おすすめされた全レースに100円ずつ均一に賭け続けた場合の成績です。レースを選んだり、金額を調整したりしていない、**完全に機械的な運用**の結果です。

## この数字がどれだけすごいのか

### ボートレースの「壁」を超えている

ボートレースの控除率（テラ銭）は**約25%**です。つまり、何も考えずにランダムに買い続けると、理論上の回収率は**75%前後**に収束します。

| 買い方 | 期待回収率 |
|--------|-----------|
| ランダム購入 | 約75% |
| 一般的な予想サイト | 70〜90%（※多くが公表せず） |
| BoatAI通常予想（スタンダード） | 88〜94% |
| **BoatAI 今日のおすすめ** | **104%** |

「今日のおすすめ」は、25%の控除率を**逆転**して利益を出しているということです。

### 2,577レースという母数の意味

「たまたまじゃないの？」という疑問は当然です。

統計学的に、サンプル数が多いほど結果の信頼性は高まります。**2,577レース**という母数は、偶然の偏りではなくパターンの再現性を示すのに十分な規模です。

参考までに：
- コインを2,577回投げて「表が53%以上出る確率」は約1.2%
- つまり、**98.8%の確率で偶然ではない**と言える水準です

## なぜ回収率100%を超えられるのか

### 「全レース予想」と「厳選」の違い

BoatAIの通常予想は**全レース**を対象にしています。当然、予測しやすいレースもあれば、難しいレースもあります。全体で平均すると、回収率は100%に届きません。

「今日のおすすめ」は発想が違います。

**過去のデータを徹底的にマイニングして、統計的に回収率100%を超える条件を発見し、その条件に合致するレースだけを抽出する**

というアプローチです。

つまり：
- 通常予想 → 全レースに参加して平均点を取る
- 今日のおすすめ → **勝てる試合だけを選んで参加する**

この「選ばない勇気」が、100%超えの源泉です。

### データマイニングが見つける「歪み」

ボートレースのオッズは、大勢のファンの購入傾向で決まります。しかし、人間の判断には偏りがあります。

- ある会場の特定条件下で、実力以上にオッズが高くなる（過小評価される）パターン
- 特定のレース番号や時間帯で、統計的に有利な賭け方が存在するパターン

こうした「オッズの歪み」は、数千レースのデータを機械的に分析して初めて見えてくるものです。人間の直感や経験だけでは気づきにくい、**データだけが教えてくれるパターン**です。

## 正直に伝えておくべきこと

### 回収率104%は「大儲け」ではない

2,577レース × 100円 = 257,700円の投資に対して、利益は11,470円です。率直に言って、これだけで生活できる金額ではありません。

ただし重要なのは、**プラス収支を維持できている**という事実です。多くの予想サービスが回収率80〜90%台で「優秀」とされる中、100%を超えていること自体が希少です。

### 全ルールが常に勝てるわけではない

「今日のおすすめ」には複数のパターン（ルール）が含まれています。その中には：

- 運用開始以来ずっと好調なルール
- 一時的に不調になっているルール

が混在しています。全体として104%を維持していますが、**個別のルールレベルでは波があります**。

### 過去の実績は将来を保証しない

統計的に有効なパターンでも、ボートレースの環境変化（選手の入れ替わり、場の改修、季節変動など）により、将来的に成績が変動する可能性はあります。

BoatAIでは、ルールの有効性を定期的に再検証し、成績が悪化したルールは見直しを行っています。
