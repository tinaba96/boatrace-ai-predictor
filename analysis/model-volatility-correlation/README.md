# 荒れ度と各モデルの予測精度の相関分析

## 概要

このディレクトリには、ボートレース予想モデル（スタンダード、本命、穴狙い）の予測精度と、レースの荒れ度との相関を数学的に分析した結果が含まれています。

**分析対象**: 単勝的中率、3連単的中率、3連単回収率

## 分析内容

### 主要な発見（単勝）

1. **的中率低下率の比較**（低荒れ度 → 高荒れ度）
   - スタンダード: 59.65% → 31.14%（47.8%減）
   - 本命: 43.86% → 27.84%（36.5%減）
   - **穴狙い: 26.90% → 22.34%（16.9%減）** ← 最も安定

2. **統計的有意性（Z検定）**
   - スタンダード: Z値 = 5.922（有意）
   - 本命: Z値 = 3.468（有意）
   - **穴狙い: Z値 = 1.092（有意でない）** ← 荒れ度に関わらず一定の性能

3. **相関係数**
   - スタンダード: -0.9458（非常に強い負の相関）
   - 本命: -0.8247（強い負の相関）
   - 穴狙い: -0.7622（比較的弱い負の相関）

### 主要な発見（3連単）

1. **3連単的中率と荒れ度の相関**
   - スタンダード: -0.4415（中程度の負の相関）
   - **本命: -0.0988（ほとんど相関なし！）** ← 重要
   - 穴狙い: -0.2607（弱い負の相関）

2. **3連単回収率と荒れ度の相関**
   - スタンダード: -0.3844（弱い負の相関）
   - **本命: -0.0903（ほとんど相関なし！）** ← 重要
   - 穴狙い: -0.3098（弱い負の相関）

3. **荒れ度別3連単回収率**
   - 低荒れ度: スタンダード 59.18%、穴狙い 51.11%
   - 中荒れ度: **穴狙い 28.72%**（最高）
   - 高荒れ度: 穴狙い 9.16%（最高）
   - **特殊区間（20-29）**: スタンダード 131%、穴狙い 116%（プラス収支！）

### 実用的な結論

#### 単勝
- **低荒れ度（0-33）**: スタンダードモデルが圧倒的に優位（的中率59.65%）
- **中荒れ度（34-66）**: スタンダードモデル推奨（的中率38.19%）
- **高荒れ度（67-100）**: スタンダードが依然優位だが、穴狙いの安定性も魅力

#### 3連単
- **低荒れ度（0-33）**: スタンダードモデル（回収率59.18%）
- **中荒れ度（34-66）**: **穴狙いモデル推奨**（回収率28.72%が最高）
- **高荒れ度（67-100）**: 穴狙いモデル微推奨（回収率9.16%）
- **特殊区間（20-29）**: スタンダードまたは穴狙い（プラス収支期待可能）

**重要な発見**:
- 単勝: 穴狙いモデルは荒れ度による性能変化が統計的に有意ではない
- 3連単: **本命モデルは荒れ度の影響をほとんど受けない**（相関係数-0.10）

## ファイル構成

### スクリプト

#### 単勝分析
- **analyze-model-volatility-correlation.cjs**
  - 単勝的中率の相関分析を実行
  - 3区間分析（低/中/高）と10区間詳細分析
  - ピアソンの相関係数を計算

- **visualize-volatility-correlation.cjs**
  - ASCIIグラフによる視覚化
  - 相対パフォーマンス分析
  - 統計的有意性の検証（Z検定）
  - 推奨モデルの精度検証

#### 3連単分析
- **analyze-trifecta-correlation.cjs**
  - 3連単的中率と回収率の相関分析
  - 単勝との比較分析
  - 荒れ度別の回収率計算

### レポート

#### 単勝分析
- **model-volatility-correlation-report.txt**
  - 基本統計レポート（単勝）
  - 相関分析結果
  - 詳細統計（10区間）
  - 自動生成された考察

- **model-volatility-visualization-report.txt**
  - ASCIIグラフによる視覚化
  - 相対パフォーマンス分析
  - 統計的有意性の詳細
  - 推奨モデル精度検証

#### 3連単分析
- **trifecta-correlation-report.txt**
  - 3連単的中率と回収率の統計レポート
  - 単勝 vs 3連単の比較
  - 荒れ度別の回収率分析

### データ

- **model-volatility-correlation-data.json**
  - 単勝分析結果のJSON形式データ
  - 基本統計、詳細統計、相関係数を含む

- **trifecta-correlation-data.json**
  - 3連単分析結果のJSON形式データ
  - 的中率、回収率、相関係数を含む

## 使用方法

### 分析の実行

```bash
# 単勝的中率の相関分析
node analyze-model-volatility-correlation.cjs

# 視覚化と詳細分析
node visualize-volatility-correlation.cjs

# 3連単的中率と回収率の相関分析
node analyze-trifecta-correlation.cjs
```

### データ要件

- `../../data/predictions/` ディレクトリに予測データが必要
- ファイル形式: `YYYY-MM-DD.json`
- 各レースには `volatility`、`predictions`、`result` データが必要

## 分析対象データ

- **期間**: 2025年12月
- **総レース数**: 2,925レース
- **データソース**: `data/predictions/` ディレクトリ

## 技術的詳細

### 相関分析

- **手法**: ピアソンの積率相関係数
- **評価指標**:
  - 単勝: トップピック的中率
  - 3連単: 3連単的中率、回収率（100円賭け）
- **区間分割**:
  - 基本分析: 3区間（0-33、34-66、67-100）
  - 詳細分析: 10区間（10刻み）

### 統計的検定

- **手法**: 2標本の割合の差のZ検定
- **有意水準**: 95%（α = 0.05）
- **臨界値**: Z = ±1.96
