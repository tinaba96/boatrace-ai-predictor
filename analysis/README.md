# 分析レポート

このディレクトリには、ボートレース予想システムの各種分析結果が格納されています。

## ディレクトリ構成

### model-volatility-correlation/

荒れ度と各予測モデル（スタンダード、本命、穴狙い）の予測精度の相関分析

**主要な発見**:
- スタンダードモデル: 荒れ度が高いと性能が大幅に低下（47.8%減）
- 本命モデル: 荒れ度の影響を受けやすい（36.5%減）
- **穴狙いモデル**: 荒れ度への耐性が最も高い（16.9%減）

**推奨戦略**:
- 低荒れ度: スタンダードモデル（的中率59.65%）
- 高荒れ度: スタンダード推奨だが、穴狙いの安定性も魅力

詳細: [model-volatility-correlation/README.md](./model-volatility-correlation/README.md)

## 分析の追加方法

新しい分析を追加する場合は、以下の構成で作成してください：

```
analysis/
└── [分析名]/
    ├── README.md              # 分析の概要と結果
    ├── [スクリプト].cjs       # 分析スクリプト
    ├── [データ].json          # 分析結果データ
    └── [レポート].txt         # テキストレポート
```

各分析ディレクトリには必ずREADMEを含め、以下の情報を記載してください：

- 分析の目的
- 主要な発見
- 使用方法
- データ要件
- 生成日時
